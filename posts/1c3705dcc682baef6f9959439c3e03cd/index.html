C语言实现十进制转换为十六进制

        编写了C语言程序，实现了十进制数到十六进制数的转换

代码：

#include <stdio.h>
int main()
{
    int a, i = 0;
    char s[200];
    printf("请输入十进制数字：\n");
    scanf("%d", &a);
    while (a)
    {
        if (a % 16 >= 10)
            s[i] = a % 16 + 55;
        else
            s[i] = a % 16 + 48;
        i++;
        a = a / 16;
    }
    printf("十进制转十六进制的数字是：\n");
    for (i = i - 1; i >= 0; i--)
        printf("%c", s[i]);
    return 0;
}
