Ubuntu 默认安装的vi超级难用

在插入状态下，按方向键出来的结果竟然是大写的字母ABCD，这是因为在ubuntu中其实没装vi,只装了vim-tiny，在系统上，vi仅仅是vim的一个别名。

这时候需要自己安装完整版的 vim

sudo apt-get install vim

将全新安装2个包，完成即爽！
