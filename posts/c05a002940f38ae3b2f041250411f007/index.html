python if not的用法

参考  python if not的用法 - 云+社区 - 腾讯云

if not (1 > 2):        #如果()中的表达式为假
   print("hahaha")    #1 > 2结果是假,所以执行hahaha
else:
   print("hihihi")

Output:
--------
hahaha
--------


if not 1:              #1为真,并不为假,因此不执行hahaha,执行hihihi
   print("hahaha")
else:
   print("hihihi")

--------
hihihi
--------
