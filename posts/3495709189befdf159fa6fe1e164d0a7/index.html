Nginx 显示中文乱码解决

中文乱码，心里瞬间一万条草泥马奔腾。在网上查了一下资料，发现需要修改nginx的server的配置内容，增加一行：charset utf-8;

详情如下：

    upstream you.domainName.com {
        server 127.0.0.1:8081;
    }
 
    server {
      listen      80;
      server_name  you.domainName.com;
      charset utf-8;
 
      location /examples {
          return 403;
        }

然后，重启Nginx服务。最后，在浏览器上Ctrl+F5，刷新，一切正常！如下图
