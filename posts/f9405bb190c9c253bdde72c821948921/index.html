C++题目：反向输出四位数

反向输出四位数

题目描述

给定一个四位数的正整数 n = a b c d ‾ n=\overline{abcd} n=abcd，请反向输出 n n n，即输出 d c b a ‾ \overline{dcba} dcba。

输出时保留前导 0 0 0。

输入格式

输入共一个正整数 n n n。

输出格式

反向输出正整数 n n n。

样例 #1

样例输入 #1

1230

样例输出 #1

0321

提示

对于 100 % 100\% 100%的数据, 1000 ≤ n ≤ 9999 1000 \le n \le 9999 1000≤n≤9999。

答案

#include<bits/stdc++.h>
using namespace std;
int main(){
    int x,a,b,c,d;
    cin >> x;
    d = x%10;
    c = (x/10)%10;
    b = (x/100)%10;
    a = x/1000;
    cout << d << c << b << a;
    return 0;
}

#include<bits/stdc++.h>

{
    string s;

    }
