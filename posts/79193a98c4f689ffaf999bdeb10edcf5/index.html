c语言编写加法

首先创建项目

int main() //主函数
{
int x,y,sum; //定义变量
printf (“请输入两个数 x和y：\n”) ; // /n 表示换行
scanf ("%d %d",&x,&y); //scanf的用法是：scanf(“格式控制字符串”,输入参数一，输入参数二)；
// %d 十进制有符号整数 &是一个取地址符,&x表示变量x的地址
sum=x+y; // 改变符号即可加减乘除
printf (“结果为%d\n”,sum); //打印sum的值
}

编译运行

输入数值
scanf ("%d %d",&x,&y); 注意 %d 和%d 之间要有间隔
输入完成 回车 即可打印出结果
