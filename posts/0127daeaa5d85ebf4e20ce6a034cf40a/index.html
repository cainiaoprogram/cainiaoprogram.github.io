rollup-plugin-copy 配置

rollup-plugin-copy 是一个用于 Rollup.js 打包构建工具的插件，用于将指定的文件或目录从一个位置复制到另一个位置。

以下是 rollup-plugin-copy 的一些常用配置选项：

  • targets：将文件复制到的目标位置。可以是一个字符串或字符串数组，也可以是一个包含 src 和 dest 属性的对象数组。
  • hook：生命周期钩子，例如 buildStart、buildEnd、generateBundle、writeBundle 等。
  • flatten：是否将复制的文件都放置在目标目录的根目录下，而不是在整个目录结构中保留原始文件的路径结构。
  • verbose：是否输出详细的日志信息。
  • copyOnce：是否只复制尚未复制过的文件。

下面是一个简单的 rollup-plugin-copy 配置例子：

import copy from 'rollup-plugin-copy';

export default {
  // ...
  plugins: [
    copy({
      targets: [
        { src: 'public/index.html', dest: 'dist' },
        { src: 'public/assets/**/*', dest: 'dist/assets' }
      ],
      verbose: true
    })
  ]
}
