ubuntu 16.04 /dev/sda1 满了如何清理

  1. 使用如下命令侦察

df -hl

 也可以直接输入以下命令，进入占用比较大的目录看看情况：

sudo du --max-depth=1 --human-readable * | sort --human-numeric-sort   

* 处 换成相应的目录，一般从 / 开始

 

  2. 找到相应的文件就开始清理了：

文件：sudo rm -f 

文件夹： sudo rm -f 

 

3. sudo reboot 走起
