react CodeMirror react-codemirror2 改变高度

CodeMirror有个默认的高度300px，尝试在less文件修改，没有修改成功

/deep/ .CodeMirror {
    height: auto;
  }

然后用下面这种方法修改成功：

const codemirrorDom = document.getElementsByClassName('CodeMirror')[0]
codemirrorDom.setAttribute("style","height: auto")
