Java中有没有类似C#中的区域注释？

 

C#中的区域注释可以使用#region和#endregion，将代码折叠后，代码结构看起来很清晰，便于阅读，如下：


Console.WriteLine(">>>>>>>>>XXX<<<<<<<<<");

#region 示例代码区域
Console.WriteLine(">>>>>>>>>XXX<<<<<<<<<");
if (true)
{
    Console.WriteLine(">>>>>>>>>XXX<<<<<<<<<");
}
else
{
    Console.WriteLine(">>>>>>>>>XXX<<<<<<<<<");
}
#endregion

Console.WriteLine(">>>>>>>>>XXX<<<<<<<<<");

其实Java中也是有类似的：


System.out.println(">>>>>>>>>XXX<<<<<<<<<");

//region 示例代码区域
System.out.println(">>>>>>>>>XXX<<<<<<<<<");
if (true) {
    System.out.println(">>>>>>>>>XXX<<<<<<<<<");
} else {
    System.out.println(">>>>>>>>>XXX<<<<<<<<<");
}
//endregion
    
System.out.println(">>>>>>>>>XXX<<<<<<<<<");
