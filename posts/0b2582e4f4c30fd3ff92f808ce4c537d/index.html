Linux中JAVA的安装配置

1、简介
JavaSE：标准版
JavaME：微型版
JAVAEE：企业版

JAVA版本：JDK6u3 也称为java6
6u3 -> Version 6 update 3 版本号6,第三次更新版
sdk -> Software Develop Kit 软件开发包
ml -> Multi Language 多语言版本,安装包安装程序支持多种语言(非ML版本只有英文)

官网：http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

2、安装配置

download
nohup wget http://download.oracle.com/otn-pub/java/jdk/8u144-b01/090f390dda5b47b9b721c7dfaa008135/jdk-8u144-linux-x64.tar.gz?AuthParam=1501346908_907cdcc5a265a61a3c88ae2f31c0be32 &
install
tar xfvz jdk-8u144-linux-x64.tar.gz
configuration
echo "JAVA_HOME=/usr/local/jdk1.8.0_144" >> /etc/profile
echo 'export PATH=$JAVA_HOME/bin:$PATH' >> /etc/profile
echo 'CLASSPATH=.:$JAVA_HOME/lib:$JAVA_HOME/lib/tools.jar' >> /etc/profile
source /etc/profile

echo 'Enter "java -version" to confirm the configuration success'
