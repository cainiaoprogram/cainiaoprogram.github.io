AIX7.1安装Oracle19c报错Error in invoking target 'irman ioracle… 解决方案

在AIX7.1上安装Oracle19c报错：Error in invoking target ‘irman ioracle idrdactl idrdalsnr idrdaproc’ of makefile ‘/oracle/app/oracle/product/19.5.0/dbhome_1/rdbms/lib/ins_rdbms.mk’. See '/tmp/InstallActions**/**.log’ for detail

原因：
查看日志得知有0711-715 ERROR,如下：
INFO：
ld: 0711-715 ERROR: File ***($ORACLE_HOME)/lib//libjavavm19.a cannot be processed.
The file must be an object file,an import file,or an archive.
判断是libjavavm19.a文件的问题

解决步骤：
拷贝$ORACLE_HOME/javavm/jdk/jdk8/lib下面的该文件覆盖$ORACLE_HOME/lib下的文件
命令：
cp $ORACLE_HOME/javavm/jdk/jdk8/lib/libjavavm19.a $ORACLE_HOME/lib/libjavavm19.a
