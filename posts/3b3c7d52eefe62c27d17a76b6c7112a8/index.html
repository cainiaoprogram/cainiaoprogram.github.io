[求助]使用scanf时，char型的正确输入方法应该是如何？

首先是第一段代码

#include

#include

main()

{

char c1,c2;

scanf("%c,%c",&c1,&c2);

printf("c1='%c',c2='%c'\n",c1,c2);

system("pause");

}

建立c1,c2为字符型变量，输入c1,c2的值后输出c1,c2。

输入：

a,b

输出：

c1='a',c2='b'

以上没有问题。

#include

#include

{

}建立了两个整型变量 a,b。并先用scanf输入a,b的值。然后再输入c1,c2的值。

输入：

12，34

c1='

',c2='f'

12，34
