k进度 (k-progress)

A Vue plugin, linear progress bar.

Vue插件，线性进度条。

View Demo 查看演示 View Github 查看Github

📦安装 (📦 Install)

npm install -S k-progress
# or
yarn add k-progress

🔨开始开始 (🔨 Begin Start)

// main.js
import KProgress from 'k-progress';
Vue.component('k-progress', KProgress);
