redis跳表的层数

redis跳表在插入一个元素时,先随机出一个该元素的所在的层数

int random_level()  
{  
    K = 1;  
    while (random(0,1))  
        K++;  
    return K;  
}

然后在将该元素加入第K层的链表中.
