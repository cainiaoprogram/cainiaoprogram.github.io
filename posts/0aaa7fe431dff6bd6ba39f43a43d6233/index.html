该问题是因为(场景描述)：

build.gradle文件中导入了

dependencies {

compile 'com.github.arcadefire:nice-spinner:1.3.3'

}

后来由于要改这个控件里的代码，故将源码导入到项目中，但是引用代码的inport未修改，从而报了此错。

问题排查方法：

在android-studio中的Terminal 敲入命令

RubertdeMacBook-Pro:oms Rubert$ ./gradlew compileDebugJavaWithJavac
