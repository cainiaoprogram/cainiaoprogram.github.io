GROMACS知识问答

本文来源：“分子动力学”公众号

链接：https://mp.weixin.qq.com/s/lg7tuIcsvG1j53egha70yg

问：我在用gromacs计算rmsf时出现了这样一个警告，我的结构也没有跑出水盒子，为什么会这样？有什么解决办法吗？


答：你选的这个group刚好在盒子边界上，继续模拟，看它会不会离开边界

问：意思是延长模拟时间吗？我这个体系比较大，一天只能运行1ns？是因为我这个水盒子和 pbc 显示不重合的原因吗？是因为这个 cell origin设置的问题吗？这个值我是按照教程输的立方水体的中心，还是我周期性边界条件设置的太小？gromacs中a run input file 指的是tpr文件吗？

答：你这个设定不是gromacs啊？从你给的盒子来看，你建模有问题，你把盒子中点建在边界上。

问：我是用NAMD 做的模拟，建模是根据NAMD教程来的，然后想用gromacs来处理数据

答：你把这个教程发我看看，一般来说namd模拟结果，vmd就能分析。不需要再用gromacs的工具

问：我在网上看到有人讲把dcd文件转化为trr文件时要修正pbc，namd教程中pbc的值和cell origin的值，就是加了水盒子之后输出的一个值

答：模拟过程没什么差别，但是各个软件保存轨迹的方式不一样。显然你转换的过程中，盒子的圆点不在（0，0，0）这个地方。问题可能出在两地地方，

1、你建盒子的时候，就不在0点

2、转换轨迹的过程中

问题应该是你建模导致的，你要就直接用 gromacs重新跑好了，gromacs的计算效率比NAMD要快很多
