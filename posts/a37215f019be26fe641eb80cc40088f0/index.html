以root用户创建docker容器

在docker run完毕之后进去发现是非root用户，这样的话建议docker run是时候用root身份操作

docker run -d  --name jenkins-test -u 0  -it 129f7dfd60d8 bash

这样再进去就是root用户了

docker exec -it jenkins-test bash
