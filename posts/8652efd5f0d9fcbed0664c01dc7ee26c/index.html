Python3 将代码打包成exe程序并添加图标

环境安装

  1. Python 3（略）
  2. pyinstaller
pip install pyinstaller

打包exe程序

命令：

pyinstaller -F -w *.py

说明：

  1. *是.py的全部路径
  2. 此命令在命令行窗口执行
  3. -F是打包单一文件，-w是运行时不弹出黑窗口
  4. 打包成功，会显示exe程序的位置，找到运行

给exe程序添加图标

命令：

pyinstaller -F -w -i *.ico *.py

说明：

  1. 需要准备一张.ico后缀名的图片（可自行网上下载）
  2. 需要注意的是，需要移动exe程序的位置，才可以显示图片

常用pyinstaller 命令

  • -i 给应用程序添加图标
  • -F 指定打包后只生成一个exe格式的文件
  • -D –onedir 创建一个目录，包含exe文件，但会依赖很多文件（默认选项）
  • -c –console, –nowindowed 使用控制台，无界面(默认)
  • -w –windowed, –noconsole 使用窗口，无控制台
  • -p 添加搜索路径
