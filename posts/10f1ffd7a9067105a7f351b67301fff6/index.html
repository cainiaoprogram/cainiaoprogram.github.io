ubuntu找不到ifconfig

Command 'ifconfig' not found, but can be installed with:

sudo apt install net-tools

出现这样的提示就是说没有找到ifconfig的命令，可以进行安装，按照提示中的命令安装相关的工具包 sudo apt install net-tools

安装完成就可以使用ifconfig。

unbuntu使用yum

  1. 切换为root用户
sudo -s
  1. 检测是否安装了build-essential程序包,输入命令:
apt-get install build-essential

（安装过）
