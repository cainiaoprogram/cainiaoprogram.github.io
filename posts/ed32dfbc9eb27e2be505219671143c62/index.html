类型的转换

double 转换成 int

Scanner in = new Scanner(System.in);
int foot;
double inch;
System.out.println((int)((foot+inch/12)*0.3048*100));

int 转换成 double

Scanner in = new Scanner(System.in);
int foot;
int inch;
System.out.println((foot+inch/12.0)*0.3048);
