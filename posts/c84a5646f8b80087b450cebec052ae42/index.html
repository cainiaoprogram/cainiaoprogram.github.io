sv- typedef

一个文件中多个类引用，一般是先定义被调用的类，

class  class_0;

endclass

class class_1;
       class_0 cl_0;
 endclass

如果不想先定义被调用的类，sv 又没有类似c 的.h 文件存储类声明，编译时从上到下解析，就会遇到类未定义，sv 就使用tydef 来做声明，先通过编译。

typedef class class_0;

class class_1;
       class_0 cl_0;
 endclass


class  class_0;

endclass

同样可以解决互相调用的情况。

typedef class class_0;

class class_1;
       class_0 cl_0;
 endclass


class  class_0;
      class_1 cl_1;
endclass
