查找SAP表空间占用空间最大的几张表

查SAP系统表空间里最大几张表，我们一般用DB02–> Space -->Segments --> Overview，点击Top Size菜单查看，但是并没有看到这个表空间里真实的最大几张表。只好用查SQL的方法去看。
SQL> select * from (select segment_name, bytes/1024/1024 from dba_segments where tablespace_name=‘PSAPSR3’ order by bytes desc) where rownum <= 5;

查看lob对应的表：
select table_name,column_name,segment_name,index_name from user_lobs where segment_name = ‘lob的名字’;
