问题产生

@Service
public class A{
	@Autowire
	B b;
}
@Service
public class B{
	@Autowire
	A a;
}

解决方案

我们可以把两个类公用的方法单独写一个类，然后两个类引入公共bean就好了

@Service
public class A{
	@Autowire
	C c;
}
@Service
public class B{
	@Autowire
	C c;
}

@Service
public class C{
}
