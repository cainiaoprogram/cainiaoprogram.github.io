函数:判断一个整数是否既是5又是7的整倍数

编写一个函数fun，函数的功能是：判断一个整数是否既是5又是7的整倍数。若是，输出yes，否则输出no。在主函数中输入任意一个正整数，调用fun函数，输出该数对应的结果

#include<stdio.h>
int fun(int n)
{
if(n%5==0 && n%7==0)
return 1;
else
return 0;
}
int main()
{
  int n;
  scanf("%d",&n);
  if(fun(n)==1)
  printf("yes");
  else
  printf("no");
  return 0;
}
