ubunt18.04 使用SimpleScreenRecorder进行屏幕录像

Simple Screen Recorder

优点：

        录制OpenGl应用程序不需要额外下载解码器；

        低配电脑自动降帧；

        随用随录，操作任性；

        声音扩展性功能；

        可选择录制编码器；

        显示录制文件大小，心里有底。

安装：

sudo add-apt-repository ppa:maarten-baert/simplescreenrecorder # 添加源
sudo apt-get update # 更新源
sudo apt-get install simplescreenrecorder # 安装
