引言

有时候，和众多编程语言一样，我们需要在 Script Prompt 中打印一些我们已经得到的数据，这样可以方便我们调试代码和查看代码中是否有错误。关于在 Script Prompt 中打印数据，Lumerical Script 提供了两种方法。

正文

方法1

mode1 = 1.677360;
?mode1;

方法2

mode1 = 1.677360;
print(mode1);
