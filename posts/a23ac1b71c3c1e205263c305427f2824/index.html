微信小程序页面如何滚动到底部

在需要滚动的页面根节点定义一个id="page"

<view id="page">
    <text>需要滚到的页面</text>
</view>

 点击事件触发以下方法

wx.createSelectorQuery().select('#page').boundingClientRect(rect => {
  wx.pageScrollTo({
    duration: 500,    // 动画时长
    scrollTop: rect.height
  })
}).exec()
