安装

依赖

有些 apt-get install 就可以，readline 和 termcap 需要下载源码安装

编译安装仿真器

cd EpicSim
mkdir build
cd build
cmake …
sudo make install
中间会遇到问题 缺少lib lbz2
$ sudo apt-get install zlib1g-dev
$ sudo apt-get install bzip2*
还有找不到文件parse.hh 的编译报错，需要修改成parse.h 就可以了，文件名写错了。
