通过ZIP下载仓库代码后关联远程仓库分支

在项目更目录 git bash here

1、git init

2、git remote add origin xxxx（远程仓库地址）

3、git add .

4、git commit -m 'init'

5、git branch -a （查看所有分支，目前只有master）

6、git pull origin dev （从分支拉取代码）

7、git branch -a （现在就会出现 master 和 dev 两个分支）

8、git checkout -b dev origin/dev （切换分支）

9、git branch -a （dev分支就是选中状态）
