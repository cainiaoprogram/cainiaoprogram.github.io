ubuntu安装make

首先，我们安装最新版ubuntu-make。


# 将ubuntu-make加入apt软件仓库的列表
sudo add-apt-repository ppa:ubuntu-desktop/ubuntu-make


# 升级apt的软件仓库，为了加入刚才的ubuntu-make
sudo apt update


# 安装ubuntu-make
sudo apt install ubuntu-make
