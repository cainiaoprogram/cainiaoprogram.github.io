linux下miniconda卸载

conda是一个开源的包、环境管理器，可以用于在同一个机器上安装不同Python版本的软件包及其依赖，并能够在不同的Python环境之间切换，Anaconda包括Conda、Python以及一大堆安装好的工具包，比如：numpy、pandas等，Miniconda包括Conda、Python。

卸载步骤

  1. 删除miniconda安装目录
    rm -r miniconda3/
  2. 注释 bashrc 中 conda initialize 代码
    vim ~/.bashrc 
# !! Contents within this block are managed by 'conda init' !!
__conda_setup="$('/opt/module/miniconda3/bin/conda' 'shell.bash' 'hook' 2> /dev/null)"
if [ $? -eq 0 ]; then
    eval "$__conda_setup"
else
    if [ -f "/opt/module/miniconda3/etc/profile.d/conda.sh" ]; then
        . "/opt/module/miniconda3/etc/profile.d/conda.sh"
    else
        export PATH="/opt/module/miniconda3/bin:$PATH"
    fi
fi
unset __conda_setup
# <<< conda initialize <<<
  1. 删除 .condarc 相关文件
    rm -rf ~/.condarc
    rm -rf ~/.conda
  2. 刷新环境
    source ~/.zshrc
    或者退出当前ssh连接会话
