SQL server的timestamp类型在查询时，可以直接where转换后的数字

例如timestamp为0x00000000000065A6
CONVERT(BIGINT,timesta)为26022
可以直接where timesta=26022

SQL Server 2014 - 12.0.2000.8 版本
