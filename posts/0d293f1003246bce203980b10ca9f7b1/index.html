适用于 iOS、Android 和 Windows 设备的移动设备管理

随着企业环境中移动设备的数量不断增加，详细检查访问您企业资源的移动设备变得至关重要。统一终端管理软件 Desktop Central 可帮助您从一个中心位置配置和保护您的移动设备，其专为简化桌面和移动设备管理而生。
我们还提供了 Mobile Device Manager Plus，独立的企业移动设备管理解决方案，可以在本地以及云端使用。如果您正在寻找管理移动设备（无论是公司所有设备还是个人设备）的解决方案，请尝试免费使用 Mobile Device Manager Plus。

Desktop Central 的移动设备管理功能

1.移动应用程序管理
设备管理不会仅仅只是配置策略、检索资产信息和保护设备。应用程序管理与设置员工的移动设备一样重要。
使用 Desktop Centrel，您可以：
创建您独有的企业应用目录。
管理和分发内部和第三方应用程序。
与 Apple 的批量购买计划 (VPP) 和 Google 的 Play for Work集成，允许简单分发商业应用程序。
移动应用程序的黑名单和白名单。
审计您的应用程序清单。
了解关于管理移动应用程序的更多信息，请单击此处。
2.移动安全管理
没有两家企业是一样的——执行严格的策略以满足您特定的安全需求。
使用 Desktop Central 的移动安全管理功能，您可以：
加强设备密码以防止未经授权的访问。
远程锁定设备以防止设备滥用、丢失或被盗。
使用地理位置跟踪，实时跟踪设备。
彻底擦除设备的数据。
执行公司擦除以仅删除公司数据，保持个人数据不变。此功能在自带设备办公 (BYOD) 环境中非常实用，在此环境中，员工可以在其个人设备上访问公司数据。
要了解关于保护移动设备的更多信息，请单击此处。
3.配置文件管理
通过在移动设备上配置和实施策略来保护公司资源。
使用 Desktop Central，您可以为不同的部门或角色创建和配置策略与配置文件。
使用 Desktop Central 的配置文件和策略管理功能，您可以：
配置关于访问企业资源的策略和配置文件。
限制相机、YouTube、浏览器等应用程序的使用。
管理对公司帐户的访问，包括电子邮件、Wi-Fi、以及 ××× 帐户。
根据部门或位置创建一个逻辑设备组。您还可以创建组来区分公司设备和员工的个人设备。应用策略或限制并将应用程序分发给组中的所有设备。
4.移动内容管理
当员工使用移动设备进行工作时，他们需要在其设备上访问公司资源。
Desktop Central 允许管理员远程共享文件到员工的设备，所有这些都不会影响安全性。
使用 Desktop Central，您可以：
创建一个内容存储库来存储文档。
以各种格式分发文档，包括 DOC、PDF、PPTX 等。
限制非托管设备之间的文件共享。
5.电子邮件管理
Desktop Central 让您可以通过平台集中化和 Exchange ActiveSync安全管理公司电子邮件。使用移动电子邮件管理，您可以为企业所有的设备和个人设备配置、保护和管理公司电子邮件帐户。
使用 Desktop Central，您可以：
在线设置电子邮件安全策略。
数据集中化以防止未经授权的应用程序访问电子邮件数据。
限制用户修改或删除其公司电子邮件帐户。
选择性擦除公司电子邮件帐户。
6.集中化
在 BYOD 环境中，员工可以使用他们的个人移动设备执行与工作相关的任务。但是，BYOD 管理只有在您可以有效管理这些设备并保护您的数据免受损害时才可用。
Desktop Central 为您提供了设置策略和限制以保证企业数据安全的能力。Desktop Central 允许您通过以下方式进行集中化：
基于所有权注册设备。
为 BYOD 和公司设备创建不同的组。
为 BYOD 和公司设备定义不同的策略。
在员工离开组织时擦除公司数据， 同时保持用户数据不变，以保护公司数据安全。
阅读我们的白皮书以了解如何利用 BYOD。
要了解有关 BYOD 管理的更多信息，请单击此处。
7.Kiosk模式
锁定移动设备以运行单个应用程序或所选的一组应用程序。Desktop Central 针对 iOS 和 Android 设备支持锁定功能。
使用 Desktop Central，您可以：
仅允许访问有限的应用程序。
创建一个仅含有所允许应用程序的自定义主页。
限制对任务管理器和状态栏的访问。
8.审计和报表
跟踪和分析资产信息有助于保护敏感的公司数据。Desktop Central 提供了可用于审计设备的内置报表。
使用 Desktop Central 的报表功能，您可以：
扫描设备以确保其符合公司策略。
获取有关在托管设备上运行的应用程序的详细信息。
立即或在计划时间生成预定义或自定义报表。
9.资产管理
与位于企业物理工作空间内的传统工作站不同，移动设备可以在多个地点使用，这使得管理和控制它们变得异常复杂。
使用 Desktop Central，您可以：
跟踪和分析资产信息以保护敏感的公司数据。
检索关于设备的完整信息，包括设备详细信息、证书、安装的应用程序等等。
使用内置报表，获取关于设备的完整信息。
实时远程排除移动设备故障。
10.设备注册
Desktop Central 是一个统一的终端管理解决方案，适用于多种设备环境，并且提供必要的安全协议，以防止未经授权的用户无法访问您的公司网络。
您可以手动或自动注册设备、批量注册设备，或让用户自行注册其移动设备。
Desktop Central 可提供：
自动、手动或在线 (OTA) 设备注册。
使用 CSV 文件进行移动设备的批量注册。
使用一次性密码或用户的 Active Directory 凭证进行注册验证。
