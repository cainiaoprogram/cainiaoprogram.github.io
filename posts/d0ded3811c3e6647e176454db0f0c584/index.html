服务器2012r2系统远程登陆,Windows Server 2012R2远程用户登录模式设置问题

您好，

感谢您在我们论坛发帖。

按照您的描述，您遇到的问题是在Windows server 2008 r2上可以找到的RDP——TCP的修改，在Windows server 2012 R2上并没有找到。如果我的理解有误，请您及时纠正我。

首先，根据我的理解与搜索，Windows server 2012 R2上这些界面的操作已经被移除了，因此我们只有通过组策略或者是注册表的方式来完成。

请参考以下图片，在会话主机服务器上，打开本地组策略或者注册表来完成。

注册表修改登录：

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Terminal Server]

"fSingleSessionPerUser"=dword:00000000

组策略修改登录：

计算机配置-》管理模板-》Windows组件-》远程桌面服务-》远程会话主机-》连接-》修改限制链接数量的策略，将它的值设置为最大值

同时，在Windows server 2008 r2上的某些设置在Windows server 2012 r2和以后的版本中，已经不使用了。

希望以上信息对您有所帮助。

感谢您的理解与支持。
