Slither 安装和使用

测试环境 centos7

安装 slither

sudo yum -y install python3 pip3
sudo pip3 install slither-analyzer

安装 solc-select

sudo pip3 install solc-select

使用 solc-select 安装 solc，装不了的话挂下代理

solc-select install 0.4.15
# 使用的 solc 0.4.15 版本
solc-select use 0.4.15
# 查看当前使用的 solc 版本
solc --version
