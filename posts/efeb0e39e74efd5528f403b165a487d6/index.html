int *(p)[ ]与int *p[ ]的比较

 

  1. 前者表示 p是由俩个指向int值的指针构成的函数，这种声明会创建俩个指向单个int值的指针；
  2. 后者表示 p首先和*相结合，从而创建一个指向包含俩个int值的数组指针。

例：
int *(p[4])是指针数组，数组内存放的是四个指针
int (*p)[4]是数组指针，数组内存放的整形，而p指向该数组
