IDEA设置类模板

第一步：File -----> Settings(快捷键:Ctrl + Alt +S)

第二步： 选择Editor 下的 File and Code Templates

第三步：选择右侧的Clas 在①处填写要使用的模板

/**
 * @ClassName ${NAME}  //类名称
 * @Description: 类描述
 * @Author: ${USER}	//作者
 * @CreateDate: ${DATE} ${TIME}	//创建时间
 * @UpdateUser: 更新人
 * @UpdateDate: ${DATE} ${TIME}	//更新时间
 * @UpdateRemark:  更新的信息
 * @Version: 1.0	//版本号
 */

第四步 ： 设置好之后点击OK（确定）或者Apply（应用）

第五步：新建的类上面就应用上了！

IDEA设置方法模板

第一步：找到settings 中Editor下的Live Templates

第二步：在Live Templates中点击右侧的加号，选择Template Group…

第三步：输入你要创建组的名字

我定义的是Methods,由于我己经创建过了，所以图中是Method

第四步：创建出组之后，选择右侧加号的Live Template

第五步：图中①位快捷键，②为快捷键描述，③为快捷键生成的内容，④为按键生成③的无内容

使用方法：/* + Enter 会生成③的内容

/**
 * @auther: $user$		//作者
 * @Description //TODO $Description$	//描述
 * @param: $param$	//参数
 * @return: $return$	//返回值
 * @date: $date$ $time$	//创建日期
 */
