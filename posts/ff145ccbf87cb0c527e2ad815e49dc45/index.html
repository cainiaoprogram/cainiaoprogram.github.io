HTML中的 ＜!DOCTYPE＞ 声明

HTML中的 <!DOCTYPE> 声明：

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8"> 
<title>标题</title>
</head>

<body>
内容......
</body>

</html>

几乎所有主流浏览器都支持 <!DOCTYPE> 声明。（其中包括像IE、火狐、Opera、Google Chrome、Safari等主流浏览器)。

声明位于文档中的最前面的位置，处于 标签之前。 声明本身不是一个 HTML 标签；它是用来告知 Web 浏览器页面使用了哪种 HTML 版本。

现在的前端页面开发中，基本都更新到了HTML5，HTML5 不是基于 SGML，因此不要求引用 DTD。且HTML5只规定了一种<!DOCTYPE> 声明，即上述示例代码中的
我们要养成一个添加<!DOCTYPE> 声明的习惯，确保浏览器能够预先知道文档类型。
备注：<!DOCTYPE> 声明不区分大小写。
