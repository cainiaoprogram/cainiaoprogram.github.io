python-svg转png-报错-AttributeError: module ‘PIL.ImageOps‘ has no attribute ‘exif_transpose‘

错误

  File "D:\sdk\Anaconda3\lib\site-packages\cairosvg\image.py", line 78, in image
    image = ImageOps.exif_transpose(Image.open(BytesIO(image_bytes)))
AttributeError: module 'PIL.ImageOps' has no attribute 'exif_transpose'

解决

pip install --upgrade pillow -i https://pypi.tuna.tsinghua.edu.cn/simple


然后就ok了
