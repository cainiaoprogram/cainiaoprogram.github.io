Jquery复制DIV中的内容到粘贴板

HTML代码：

<div id="myDiv">
  这是要复制的内容
</div>

<button id="copyButton">复制</button>

 jQuery代码：

$(document).ready(function() {
  $('#copyButton').click(function() {
    var content = $('#myDiv').html();
    var tempElement = $('<textarea>').val(content).appendTo('body').select();
    document.execCommand('copy');
    tempElement.remove();
    alert('内容已复制到粘贴板！');
  });
});
