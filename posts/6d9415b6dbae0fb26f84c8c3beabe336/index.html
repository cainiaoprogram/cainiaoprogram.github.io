python count（）函数

Python 元组 count() 方法用于统计某个元素在元祖,列表，字符串中出现的次数。可选参数为在字符串搜索的开始与结束位置。

 

参数

  • sub -- 搜索的子字符串
  • start -- 字符串开始搜索的位置。默认为第一个字符,第一个字符索引值为0。
  • end -- 字符串中结束搜索的位置。字符中第一个字符的索引为 0。默认为字符串的最后一个位置。

例子：

字符串：

a="hello world"

print (a.count("e",1,10)

print (a.count("e")
