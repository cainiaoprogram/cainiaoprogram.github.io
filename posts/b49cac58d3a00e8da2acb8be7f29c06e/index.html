ubuntu-server 20.04 连接wifi（热点）

学习笔记：
1.在终端执行

sudo apt update

sudo nano /etc/netplan/50-cloud-init.yaml

3.将以下代码改成如下

network:
  ethernets:
    eth0:
      dhcp4: true
      optional: true
  version: 2
  wifis:
      wlan0:
          dhcp4: true
          access-points:
              "你的wifi的ssid":
                  password: "你的密码"

4.验证

sudo netplan try

sudo netplan apply

5.再验证

sudo apt-get update
