AndroidStudio查看无用的资源文件；

1、打开需要查看的项目，选择AS上方标题栏的Analyze选项；


2、选择 Run Inspection by Name


3、在弹出框内输入 unused resources


4、筛选你需要查看的资源（像第一个：完整的项目；第二个：仅app这个Module）


5、好了，这时没有引用的资源文件就学会显示在下方；
