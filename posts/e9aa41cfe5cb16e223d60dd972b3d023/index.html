前言

就注意一点，下面这句话

np.array([1,2,3])
print(np.array([1,2,3]).shape)

既不是行向量，也不是列向量，你输出它的shape：(3,)。是这么个东西，你可以理解成就是一个维度是三的array。

一、行向量与列向量生成

1.行向量


# 1.产生行向量的方法
row_vector1 = np.array([1, 2, 3]).reshape(1, -1)  #最常用
row_vector2 = np.array([[1, 2, 3]]) #也经常用
row_vector3 = np.expand_dims(np.array([1, 2, 3]), axis=0)#很少用

2.列向量

# 1.产生列向量的方法
col_vector1 = np.array([1, 2, 3]).reshape(-1, 1)#最常用
col_vector2 = np.array([[1], [2], [3]])#最常用
col_vector3 = row_vector1.T  # 向量转置 常用
col_vector4 = np.expand_dims(np.array([1, 2, 3]), axis=1)#很少用
