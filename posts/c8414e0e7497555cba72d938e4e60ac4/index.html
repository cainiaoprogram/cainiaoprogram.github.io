No Python 3.8 installation was detected.

当我们在卸载Python时，弹出”No Python 3.8 installation was detected.“对话框，可能是卸载程序已经损坏。可以通过先修复Python程序，然后再卸载程序，最后删除在路径C:\users\123\appdata\roaming\python\python38\site-packages下安装的Python的库，以免影响新一次安装Python第三方库发生错误。
