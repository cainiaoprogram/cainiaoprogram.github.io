Windows使用bat脚本实现MySQL的启动与关闭

 

Windows下

手动启动服务

mysqld --console　　
或　　net start mysql　　

手动关闭服务

mysqladmin -uroot shudown　　
或　　net stop mysql　

bat脚本：如下

@echo off  
for /f "skip=3 tokens=4" %%i in ('sc query mysql') do set "zt=%%i" &goto :next  
  
:next  
if /i "%zt%"=="RUNNING" (  
echo 已经发现该服务正在运行，正在已经关闭服务  
net stop mysql  
) else (  
echo 该服务现在处于停止状态，正在现在开启服务  
net start mysql  
)  
exit  
pause

功能：MySQL启动着进行关闭，关闭则进行启动
