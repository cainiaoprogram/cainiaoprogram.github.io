我的世界服务器op无限圈地,我的世界手机版op圈地指令一览

我的世界手机版这款游戏中，想要玩的溜这款游戏，各种指令一定要熟悉，特别是OP的指令，那么游戏中OP怎么圈地呢？用什么指令呢？下面就请看小编为大家带来的攻略吧！

/res改为/resadmin即可 无需设置.

如:/resadmin create 创建的领地不受大小限制

/resadmin set 可直接设置当前领地权限

我的世界op圈地指令介绍：

1.首先玩家需要知道在我的世界中不同服务器里圈地的工具也不尽相同，玩家需要根据情况选取合适的工具。下面小编以线为例。一般来说线都可以通过刷蜘蛛类的怪获得。

2.想要测量出玩家圈地的面积，需要用线测量一个正方体空间内的2个对角。在地面上用线点击左键确定一个点，然后点击对角的最高点用右键点击一下。

3.输入/res select size/res select size指令来确定玩家圈的地是否满意或符合要求，同时还可以看到这块地所需要的金币数量。如下图所示。

4.如果当前圈定的地符合要求，同时玩家也对圈的地满意就可以通过输入/res select size /res create 领地名字来完成购买，领地名字玩家可以自由定义。这样就完成了我的世界游戏中的圈地操作。
