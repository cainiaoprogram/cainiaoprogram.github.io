Update指定更新数据影响行数

SQL SERVER指定更新数据影响行数：

Update Top （10）A_Message set IsRead=1 where MsgCode='201403026';

SQL SERVER中 Top （10）表示每次只更新10条数据。

MYSQL指定更新数据影响行数：

UPDATE A_Message set IsRead=1 LIMIT 10;

MySQL 中 LIMIT 10表示每次只更新10条数据。
