【MATLAB笔记】基础函数及向量

一、基础函数绘制

版本：Matlab2021a

实例1：生成向量

>> x=0:0.1:30;
>> y=sin(x).*cos(x);
>> plot(x,y)

 实例2：创建向量

>> x=linspace(0,2*pi);
>> y=sin(x)
>> plot(x,y)

 二、向量的点积

>> a=[2 4 5 3]

a =

     2     4     5     3

>> b=[3 2 3 2]

b =

     3     2     3     2

>> c=dot(a,b)

c =

    35

>> c=a.*b

c =

     6     8    15     6
