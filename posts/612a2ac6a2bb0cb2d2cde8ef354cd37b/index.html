Python：plt.imshow() 没反应 不显示图像

问题描述：

调用plt.imshow()函数之后，程序没有报错，但是并没有弹出窗口，以显示图片

import matplotlib.pyplot as plt

plt.imshow(image) # 这里image表示是一张图片

问题分析：

大家可以把plt.imshow()这个函数的作用理解为：把一张图片做一些数据处理，但仅仅是做处理而已。

如果要想把这个处理后的结果显示出来，需要调用另一个函数：plt.show()

解决方案：

import matplotlib.pyplot as plt

plt.imshow(image) # 对图片image进行数据处理
plt.show() # 将图片显示出来
