kotlin map{}和mapOf{}

map{}的作用

map 让集合中的每个元素应用给定的转换函数（transform），然后生成并返回一个新的 List<R>

val numbers = listOf(1, 2, 3, 4, 5)
//map 让集合中的每个元素应用给定的转换函数（transform），然后生成并返回一个新的 List<R>
Log.d("========map", numbers.map { it * 2 }.toString())

打印结果：[2, 4, 6, 8, 10]

maxOf {}的作用

maxOf 该函数的主要目的是在集合中找到应用 selector 函数后得到的最大值

val numbers = listOf(1, 2, 3, 4, 5)
//maxOf 该函数的主要目的是在集合中找到应用 selector 函数后得到的最大值
Log.d("========maxOf", numbers.maxOf { it }.toString())

打印结果 ： 5

demo2

data class Person(val name: String, val age: Int)
val people = listOf(Person("Alice", 15), Person("Bob", 30), Person("Charlie", 25))
val age = people.maxOf { it.age }
Log.d("========maxOf", age.toString())

打印结果：30
