461. 汉明距离

两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。

给出两个整数 x 和 y，计算它们之间的汉明距离。

注意：
0 ≤ x, y < 231.

示例:

输入: x = 1, y = 4

输出: 2

解释:
1   (0 0 0 1)
4   (0 1 0 0)
       ↑   ↑

上面的箭头指出了对应二进制位不同的位置。
通过次数83,392提交次数106,673

/**
 * @param {number} x
 * @param {number} y
 * @return {number}
 */
var hammingDistance = function(x, y) {
    
    let res = 0;

    while(x !== 0 || y !== 0) {
        const bitx = x & 1;
        const bity = y & 1;
        if(bitx !== bity) {
            res++;
        }
        x >>= 1;
        y >>= 1;
    }

    return res;
};
