aardio类的例子

论坛里面相关资料太少，这里贴一下

库需要在工程的lib目录下，在ide里面就是用户库目录，比如

my_lib

namespace my_lib{

    import console

    class MyLibClass {
        ctor(...){
            ..console.log("in ctor of mylibclass");
        }
    
        my_lib_function=function(...){
            ..console.log("in my_lib_function");
            
            return "my_lib_function";
        }
    }
}

 

main文件

import my_lib
import console

test = my_lib.MyLibClass()
info = test.my_lib_function()
console.log(info)
console.pause()
