【选择结构】判断是否为两位数

前言

之前一直没有介绍什么是选择结构，今天我们就回来了解一下。

选择结构：根据结果判断条件和控制程序。

了解了选择结构后，就可以做题了。

正文

题目描述：判断一个正整数是否是两位数（即大于等于10且小于等于99）。

输入：一个正整数，不超过1000。

输出：一行。若该正整数是两位数，输出1，否则输出0。

样例输入：51

样例输出：1

思路：使用选择结构常用的if语句和else解决。

现在就来看代码吧！

这是大佬H  Cisco的代码：

#include<iostream>
using namespace std;
int main()
{
    int a;
    cin>>a;
    if((a>=10)&&(a<=99))
        cout<<"1"<<endl;
    else cout<<"0"<<endl;
    return 0;
}

然后这是本蒟蒻的代码：

#include<iostream>
using namespace std;
int main()
{
    int n;//设n
    cin>>n;
    if(n>=10&&n<=99)//如果在10-99之间
    {
        cout<<1<<endl;
    }
    else
    {
        cout<<0<<endl;
    }
    return 0;
}

大佬设输入的数是a，我则是设的n。
