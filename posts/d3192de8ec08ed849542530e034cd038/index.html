C#使用Array.Sort对二维数组排序

二维数组需要这样定义

int[][] nums= new int[][] { new int[]{ 1, 2 }, new int[]{ 3, 4 } };

有这几种方式，表示拿每个一维数组的第二个元素比较

Array.Sort(nums, (x, y) => x[1].CompareTo(y[1]));

Array.Sort(nums, (a, b) => a[1] - b[1]);   

Array.Sort(nums, (a, b) => { return p1[1] - p2[1]; });
