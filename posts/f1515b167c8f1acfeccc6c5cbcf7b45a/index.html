我本地的环境是：

IDEA 2018.1

JDK 1.8

Gradle 6.5

看到org.jetbrains.plugins.gradle.tooling.builder.ProjectExtensionsDataBuilderImpl.buildAll，
我初步怀疑IDEA的问题，可能是IDEA版本太低了，换个最新的IDEA试了试，

安装，po jie（嘘！）一气呵成之后，重新建一个gradle项目。

IDEA中gradle配置如下：

其中，.gradle文件夹是手动创建的，里面的内容会在编译的时候自动生成，init.gradle需要手动创建，里面的内容为：

allprojects{
  repositories {
    def REPOSITORY_URL = 'http://maven.aliyun.com/nexus/content/groups/public/'
    all { ArtifactRepository repo ->
      if(repo instanceof MavenArtifactRepository){
        def url = repo.url.toString()
        if (url.startsWith('https://repo1.maven.org/maven2') || url.startsWith('https://jcenter.bintray.com/')) {
          project.logger.lifecycle "Repository ${repo.url} replaced by $REPOSITORY_URL."
          remove repo
        }
      }
    }
    maven {
      url REPOSITORY_URL
    }
  }
}

配置完成后，项目自动build，此时就build成功了：
