lotus 存储时间

千兆内网

  • 扇区大小1GB
  • 存储命令
lotus-storage-miner pledge-sector
  • Empty ---> Packing耗时2m
  • Packing ---> Unsealed 耗时0s
  • performing sector replication 执行扇区复制
    1. node remote 拷贝staging 1GB 耗时9s
    1. computation 耗时 10m
    1. node remote 拷贝 sealed 1GB 耗时9s
    1. node remote 拷贝 cache 10.25GB 耗时 1m38s
  • Unsealed ---> PreCommitting 耗时 11m
  • PreCommitting ---> PreCommitted 耗时 0s
  • PreCommitted ---> Committing 耗时 11m
  • scheduling seal proof computation 调度密封证明计算
    1. node remote 拷贝 sealed 1GB 耗时9s
    1. node remote 拷贝 cache 10.25GB 耗时 1m33s
    1. starting computation 耗时 17m
    1. node remote 拷贝 cache 2GB 耗时 20s
  • Committing ---> CommitWait 耗时 19m
  • CommitWait ---> Proving 耗时 0s

节点耗时

  • node remote 总耗时 45m
  • node miner 存储到Proving sector 总耗时 49m
