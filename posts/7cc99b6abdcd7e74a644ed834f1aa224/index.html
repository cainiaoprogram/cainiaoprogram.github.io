MySQL 启动

1. 配置环境变量：

　　将MySQL /bin目录设为环境变量

2. 将服务添加到注册表中：

　　mysqld --install(添加服务) sc delete ServiceName (删除服务)

3. 初始化服务：

　　mysqld --initialize

4. 启动MySQL

　　net start mysql

5. 进入MySQL 

　　在初始化完成之后会生成临时密码 在.err文件中

　　mysql -u root -p

6. 修改密码

　　set password for root@localhost = password('root');
