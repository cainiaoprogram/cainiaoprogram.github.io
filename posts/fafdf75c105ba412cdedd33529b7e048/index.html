Python3 chardet模块查看编码格式

chardet模块可以查看如下的编码格式


需要注意的是，如果遇到ＧＢＫ２３１２等编码的，在ｄｅｃｏｄｅ和ｅｎｃｏｄｅ时，一律使用ＧＢＫ进行编码或者解码，这是因为ＧＢＫ是其他ＧＢＫ编码的超集，向下兼容所有的ＧＢＫ编码。

下面是一个例子：

#coding=utf-8
import urllib.request
import chardet
url = 'http://www.baidu.com'

a = urllib.request.urlopen(url)

'''
chardet模块
使用该模块可以查看字符串的编码格式：chardet.detect()
'''
encode = chardet.detect(a.read())
print(encode['encoding'])

#假设存在一个a.txt的文件
f = open('a.txt', 'rb')
print(chardet.detect(f.read(100)))
