ubuntu server 20.04 连接WiFi 用netplan

#列出你的网络
ip a

/etc/netplan/下面有个YAML配置文件

修改配置文件

ubuntu@ubuntu:~$ cat /etc/netplan/50-cloud-init.yaml

network:
    ethernets:
        eth0:
            dhcp4: true
            optional: true
    version: 2
    wifis:
            wlan0:
                    dhcp4: true
                    access-points:
                            "你的wifi的ssid":
                                    password: "你的密码"
netplan try

netplan apply
