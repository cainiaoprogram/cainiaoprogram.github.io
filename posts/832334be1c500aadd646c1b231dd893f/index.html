pycharm无法输入中文注释解决办法

第一步：打开pycharm,点击 File --> Settings --> Editor --> Code Style --> File Encodings, 按照图1中红色框内的状态，设置自己pycharm相应的位置的状态,点击Apply,点击ok.
图1：

第二步：点击 File --> Settings --> Editor --> Code Style --> File and Code Templates --> Python Scripts,按照图2中红色框内的状态，设置自己pycharm相应的位置的状态,点击Apply,点击ok.
图2：

第三步：关闭pycharm，重启pycharm，打开项目添加中文注释即可。
