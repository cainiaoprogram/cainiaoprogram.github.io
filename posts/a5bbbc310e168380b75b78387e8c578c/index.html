ReadTimeoutError问题解决

使用pip install 安装库时遇到ReadTimeoutError
当使用pip安装时，经常速度很慢，而且保持很慢速度下载到最后突然这种情况实在很让人崩溃：

解决方法：
将

  • pip install 所要安装的库

换成

  • pip --default-timeout=1000 install -U 所要安装的库
