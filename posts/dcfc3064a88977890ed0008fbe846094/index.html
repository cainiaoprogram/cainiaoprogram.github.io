linux screen会话管理 断开连接恢复会话

参考：https://zhuanlan.zhihu.com/p/349322691

安装
CentOS系统执行以下命令安装：

yum install screen

Debian/Ubuntu系统执行以下命令安装：

apt-get install screen

常用命令教程
实际使用过程中，大部分用户只需要掌握以下几个基本操作就满足需求了: 会话的创建，会话的查找，会话的恢复，会话的删除

详细命令如下：

新建一个screen会话
screen -S <名字>

退出当前screen会话
键盘点击ctrl+a , 然后按d

查看所有screen会话
screen -ls

进入（恢复）某一screen会话
screen -r <会话序列号>

关闭screen会话
screen -X -S <序列号> quit
