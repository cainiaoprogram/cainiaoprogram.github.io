Nginx的405错误解决办法

nginx 配置：

 location /
   {
     root  /www/client/domain;
      index index.shtml index.html;
      error_page 405 =200 /index.shtml;
   }
或
   location ~* /*\.shtml$
   {
      root  /www/client/$domain;
      index index.shtml index.html;
      error_page 405 =200 http://$domain$request_uri;
   }
