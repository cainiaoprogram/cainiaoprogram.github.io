MySql lower_case_table_names 导致 errno:150

windows 平台 MySql 5.5 中当my.ini 中设置了：

lower_case_table_names = 0  #表名大写

 数据库引擎为InnoDb , 此时创建约束就会导致:

Error Code: 1005. Can't create table '**.#sql-15f0_**' (errno: 150)
