CentOS7安装docker服务

CentOS7安装docker服务

  • 一、下载离线安装包
  • 二、安装rpm包

之前写过一篇关于Ubuntu16.04安装docker与NVIDIA相关的文章，这次因为公司服务器使用centos，发现用之前的方式还不行，有一点不一样，所以这次自己实践后就动手记录一下。

一、下载离线安装包

docker ： 下载地址 ， 分别下载：

containerd.io-1.2.5-3.1.el7.x86_64.rpm

docker-ce-19.03.5-3.el7.x86_64.rpm

docker-ce-cli-19.03.5-3.el7.x86_64.rpm

二、安装rpm包

  1. 将下载好的三个rpm文件放置centos服务器上，如下图：
  2. 执行以下命令进行安装
[root@localhost tools]# rpm -Uvh *.rpm --nodeps --force
  1. 安装完成后，输入命令启动docker服务
[root@localhost tools]# systemctl start docker
  1. 如果启动过程中遇到：Job for docker.service canceled 问题，直接执行以下命令（参考资料： docker内网环境下离线安装完成后，启动报错Job for docker.service canceled.解决方案.）
[root@localhost tools]# yum -y install systemd-devel systemd-libs libseccomp device-mapper-libs
  1. 执行完成后，再次输入步骤3的命令，启动docker服务，即可正常使用
  2. 最后，将docker 添加到开机自启中
[root@localhost tools]# systemctl enable docker
