错误一部分

于是我看日志看到了以下一行内容

  • java.lang.IllegalStateException: Failed to execute ApplicationRunner 这是一个Java异常，表示执行ApplicationRunner失败。ApplicationRunner是Spring Boot提供的一个接口，用于在应用程序启动后执行一些初始化操作。

  • 我此时恍然大悟打开看了我写的项目源码发现了自己springboot生命周期里面写的当spring程序启动时候默认用工具类发一个请求把token存到redis中，

  • @Component
public class MyApplicationRunner implements ApplicationRunner {
//    打包时候必须启动本地redis不然会打包失败
    @Autowired
    private RedisTemplate redisTemplate;
    // 在这里执行一次性初始化操作，获取token等
    @Override
    public void run(ApplicationArguments args) throws Exception {
        //            这操作应该在启动服务时候执行一次
        String accessToken = TokenUtil.getAccessToken();
        redisTemplate.opsForValue().set("accessToken", accessToken, 110, TimeUnit.MINUTES);
    }
}

    然后我发现本地redis我已经关了，我发现问题所在以后试着启动本地redis重新打包一下竟然打包成功了，然后我确定了我的猜测是对的，然后代码中出现了如下注释

    //    打包时候必须启动本地redis不然会打包失败
    @Autowired
    private RedisTemplate redisTemplate;
