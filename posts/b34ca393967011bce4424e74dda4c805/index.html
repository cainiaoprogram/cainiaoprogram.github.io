空指针异常。

所以对于一个字符串判空，需要先判null后判断是否为空串

        String str = null;
        // 方法一
        if (str == null || ("").equals(str)) {
            // TODO
        }
        // 方法二
        if (str == null || str.length() == 0) {
            // TODO
        }

（二） 之 对象判断相等

1、临时对象存储在方法区；new 对象存储在堆里。

相同临时对象会指向同一内存，new 的对象会指向不同内存。

==：判断地址是否一致

equals：判断内容是否一致

所以要使用equals

（三）之equals使用

已知非null对象.equals(待比较对象)

建议这样使用，不要将null对象置于equals()内

（四）之List.stream().map()

.stream().map()方法在使用时先判断List对象是否为空，是否为空串，避免因空指针异常引起其他问题。
