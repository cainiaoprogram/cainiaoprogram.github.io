战争雷霆显示认证服务器不可用,战争雷霆81110007错误怎么办 代码原因解决方法...

开心的更新好战争雷霆游戏，进入游戏正在等待。啪！出了一个提示框：致命错误：81110007 认证服务器不可用

更多信息：https://support.gaijinent.com/hc/en-us/articles/200070331?code=81110007?code=81110007点开连接好全英文我去！！：

出现这样的问题应该怎么办？

小编熊北北就给大家介绍一下，81110007代码的意思是客户端损坏了，尝试修复一下。

那支持的support的内容写的是什么呢？原文如下：

Error 81110007: File Read Error. Try To Repair Installation By Launcher.

This issue is caused by corrupted game client files.

To solve the issue you need to delete local settings folders:

C:\Users\user_name\AppData\Local\WarThunder\Cache

C:\Users\user_name\Documents\My Games\WarThunder\Saves

Also, please try to use "Check Files" option in the game launcher. It can be found in the sub-menu by clicking a gear button right to graphics settings.

翻译：

错误81110007：文件读取错误。尝试通过登陆器修复安装。

此问题是由损坏的游戏客户端文件引起的。

为了解决您需要删除本地设置文件夹的问题：

C:\Users\user_name\AppData\Local\WarThunder\Cache

C:\Users\user_name\Documents\My Games\WarThunder\Saves

另外，请尝试在游戏登陆器中使用“检查文件”选项。它可以在子菜单中点击齿轮按钮右边图形设置中找到。

原因方面：这个有很多，比如不正当的关闭游戏，类似于alt+F4这种。或者游戏运行中直接关闭电源等等，或者文件清理导致游戏数据丢失都容易使客户端读取不到所需文件而产生报错。大家明白了吗？
