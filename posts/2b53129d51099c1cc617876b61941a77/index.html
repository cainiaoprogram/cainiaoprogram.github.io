jSP页面中给变量赋值方法

如果在一个jsp页面中，前面需要先给某个或者某些变量赋值，后面在某个地方调用，可以采用的方法：

第一种：

<%        int age = 30;

              request.setAttribute("age",new Integer(age));                                                      %>



第二种：

package com.test;

public class Person {
private int age;
private String name;
public int getAge() {
return age;
}
public void setAge(int age) {
this.age = age;
}
public String getName() {
return name;
}
public void setName(String name) {
this.name = name;
}
}


<jsp:useBean id="per" class="com.test.Person"/>

<jsp:setProperty name="per" property="age" value="50">

<jsp:setProperty name="per" property="name" value="某某某">
