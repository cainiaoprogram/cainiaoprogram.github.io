1分钟搭建完整laravel后台

原文：https://github.com/z-song/laravel-admin


1，composer安装laravel admin

Laravel 5.1, not maintained
composer require encore/laravel-admin "1.1.*"

Laravel 5.2
composer require encore/laravel-admin "1.2.*"

Laravel 5.3
composer require encore/laravel-admin "1.3.*"

Laravel 5.4
composer require encore/laravel-admin "1.4.*"
2，
在config/app.php   ServiceProvider中添加一下:
Encore\Admin\Providers\AdminServiceProvider::class

Then run these commands to publish assets and config：
php artisan vendor:publish --tag=laravel-admin
3，安装admin后台命令  注意：安装前必须保证数据库信息连接正确！！！ 

php artisan admin:install
4，然后就可以访问测试后台了

打开 http://你的虚拟域名xxxx/admin/在浏览器,使用后台用户名 admin 和默认密码 admin 去登陆。

ok   laravel  就可以飞起啦
