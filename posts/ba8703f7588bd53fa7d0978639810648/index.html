Error: EACCES: permission denied

前言

在安装 vue 脚手架的时候，报错 Error: EACCES: permission denied

# 安装 Vue CLI 3.x
npm i -g @vue/cli

解决

执行以下命令就可以解决了

sudo npm install -g --unsafe-perm=true --allow-root
