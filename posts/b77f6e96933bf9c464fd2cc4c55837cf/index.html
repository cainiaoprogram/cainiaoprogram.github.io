List.find方法

搜索与指定谓词所定义的条件相匹配的元素，并返回整个 List(OfT) 中的第一个匹配元素。

原型：

Public Function Find ( _
	match As Predicate(Of T) _
) As T

用法:
 c# public User GetUserBy(string userName)
        {
            return GetAll().ToList().Find(t => t.UserName == userName);
        }


 
 

  
  

   
   
vb.net:  Public Function GetUserBy(userName As String) As User
  Return GetAll().ToList().Find(Function(n) n.UserName = userName)
  End Function
