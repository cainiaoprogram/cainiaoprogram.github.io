C语言中关于函数递归的理解

 递归的概念：如果一个对象部分包含它自己,或者利用自己定义自己,则称这个对象是递归的;如果 一个过程直接或间接调用自己,则称这个过程是一个递归过程。递归的主要思考方式在于：将大事化小

我们先看一个例子

题目：输入一个无符号数，并输出这个数的每一位 

​#include"stdio.h"
void print(unsigned int n)        
{
    if(n>9)
        print(n/10);
    printf("%u ",n%10);
}
int main()
{
    unsigned int num=0;        //unsigned：无符号类型
    scanf("%u",&num);          //%u 无符号的数据类型
    print(num);
    return 0;
}
