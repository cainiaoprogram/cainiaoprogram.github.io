qemu: could not load PC BIOS ‘bios-256k.bin

在centos下

首先 yum install qemu.x86_64，不过此时QEMU仍然不可用，

会提示“qemu: could not load PC BIOS 'bios-256k.bin'”

还需要再yum install qemu-kvm.x86_64
