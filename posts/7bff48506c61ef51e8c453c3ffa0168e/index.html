simulink if 模块用法

if else控制流

这个模块与 If Action Subsystem模块一起用，可完全替代c语言if else语句。
number of inputs-指定输入端口个数，用于if else中判断的变量。
然后将要判断的逻辑按格式填写，然后按填写顺序执行程序。
当u1>0,u2>0的时候，满足u1>0条件，输出为1，下面条件则不进行判断。
