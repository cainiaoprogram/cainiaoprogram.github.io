第二篇：C语言中实现两个数的加法运算

代码展示：

//实现两个数的加法运算
#include <stdio.h>  //C语言预编译指令 
int main(){
    int a,b,sum=0;  //初始化
    scanf("%d%d",&a,&b);  //输入a和b的值
    sum=a+b;    //计算a+b，其结果赋给sum 
    printf("%d, %d,sum=%d\n",a,b,sum);
    return 0;
}
