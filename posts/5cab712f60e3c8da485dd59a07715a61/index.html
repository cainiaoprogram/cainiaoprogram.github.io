Neo4j突然运行不了，log文件显示“Neo4j Server shutdown initiated by request,Stopping...,Stopped.”解决方案

用了很久的neo4j，突然运行不了

cat一下文件/var/log/neo4j/neo4j.log
报错信息显示：

2023-12-21 15:01:56.353+0000 INFO  Neo4j Server shutdown initiated by request
2023-12-21 15:01:56.355+0000 INFO  Stopping...
2023-12-21 15:02:01.410+0000 INFO  Stopped.

在网上找了很多的解决方案都没有解决
但是最后还是解决了，下面分享一下解决方案。

  1. 进入文件目录/var/lib/neo4j/data/databases
  2. 删除这个文件目录下的文件store_lock，必要时需要管理员权限来删除
  3. 再去neo4j start
  4. 问题解决！
