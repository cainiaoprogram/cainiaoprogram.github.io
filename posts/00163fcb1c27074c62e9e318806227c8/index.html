安装emqx

docker run -d --restart=always  --privileged=true  --name hb-emqx -p 1883:1883 -p 8081:8081 -p 8083:8083 -p 8084:8084 -p 8883:8883 -p 18083:18083 emqx/emqx:latest

查看是否安装并启动成功

docker ps
