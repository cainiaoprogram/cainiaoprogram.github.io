python 列表推导式if或者多个if_else判断多种情况的列表推导式

一、 只判断一种情况的列表推导式，如判断是否是偶数：

a=[i  for i in range(20) if i %2==0]
print(a)

二、 对于多种情况获得多种结果，if else结构：

a=['零' if i==0 else '三' if i==3 else '五' if i==5 else i  for i in range(20) ]
print(a)
