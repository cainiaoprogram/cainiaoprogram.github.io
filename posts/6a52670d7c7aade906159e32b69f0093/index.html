TS无法找到模块“common.js”的声明文件

按照百度的新建common.d.ts文件并输入declare module '*.js';  依然会报错common.d.tsis not a module.

正确解决办法是:

src下的shims-vue.d.ts中添加 declare module '*.js';

// 目录文件: src/shims-vue.d.ts
declare module '*.vue' {
  import Vue from 'vue'
  export default Vue
}
declare module '*.js';// 添加这个
