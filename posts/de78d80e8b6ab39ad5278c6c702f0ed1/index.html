1、字体大小设置

【Options】->【Preferences】->【Typing】->【Per File Type】->【Screen Font】

2、行号显示

【View】->【Line Numbers】

3、快速跳转到某一行

Ctrl+G

4、在所有文件中的搜索某一函数使用情况

Ctrl+/

5、换行自动对齐

】Options】->】File type Options】，选择】Auto Indent…】弹出对话框，将】Smart Indent Options】下的两个复选框的√去掉即可

6、取消程序名后的文件路径显示

点击文件名->Windows Tab Options，进去取消勾选即可
