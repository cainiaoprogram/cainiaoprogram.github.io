sqlserver删除master数据库中的用户创建的表

最近在使用SQL Server2019，在程序中执行建表语句默认建到了master数据库中，使用SSMS(SQL Server Management studio)执行以下语句时提示：“不允许对系统目录进行即席更新”

delete from sysobjects where id = object_id('table_name') and OBJECTPROPERTY(id, 'IsUserTable') = 1
于是百度了一下，使用下列命令可以删除master数据库中的所有用户添加的表


use master;
go
sp_msforeachtable @command1="drop table ?"
go
