ivr+文件服务器,IVR产品 Extra IVR 3.0

Extra IVR 3.0 是一个基于 Windows NT 平台的支持多线路/多服务器的电话语音应答系统。它是为从低成本的只有几条电话线的语音查询系统到大型的多条E1中继线的呼叫中心应用而设计的，并包含着一个内建的管理系统。

Extra IVR 3.0 包括IVR服务器，基于网络的管理服务器和多种电话语音硬件平台并以此提供了业内领先的多种功能。

Extra IVR 3.0 产品包含IVR应用设计和系统运行两种环境。应用设计环境包含 Visual Editor 和 MiniPhone。系统运行环境包含 IVR 服务器和 IVR 网络监视器。

Extra IVR 3.0 可以被用来扩展为支持：

更多语言版本

自然语音识别

统一信息服务系统

IVR服务器

IVR 服务器是一个多线程的NT服务程序,它具有以下功能:

提供应用服务功能

不同的 IVR 通道能够运行不同的业务流程

动态内容传真

从数据库或一般格式的文档中提取动态的传真内容

基于脚本的 IVR 系统

能够以图形工具或手工编写脚本的方式定制业务流程

支持多语言

英文和中文

文本语音转换

以自然语言方式播放或将数字,日期,时间,字符等转换成语音

开放式接口

嵌入式 VBScript 和 ActiveX

ODBC, SQL

与其它应用程序网关的 Socket 接口

开放式系统结构

可与 Extra Fax 或其它传真产品相连接以提供传真功能

独立于硬件的系统设计

不需要改变任何一行代码就可以支持来自 Dialogic, Telesynergy 或其他供应商的硬件

灵活的系统配置

从几条模拟线的小型系统到多条 E1 的呼叫中心应用

IVR 网络管理器

IVR Manager 类似 IIS 管理器或 SQL 服务管理器用来监测系统状态. IVR Manager 可运行在 NT 服务相同的机器上,也可在另外一台机器上运行; 它可以用来管理多个IVR 服务器, 其功能如下:

可同时监测多个 IVR 服务器

显示实时 IVR 系统运行数据

监测 IVR 服务器上的所有电话板卡与通道

监测语音应答单元的访问时间和频率

可视化流程编辑器

Visual Editor 是 Extra IVR 3.0 系统的一个组成部分. 它以其较高的性能和广泛的适应性, 为终端用户或集成商提供了对 IVR 脚本工作流程的控制.

为了最大限度地方便用户, Visual Editor 为 IVR 脚本的编写提供了一整套可视化的图形工具. 在它的集成开发环境中, 你可以执行定义一套语音文件, 定义和初始化一些变量, 定义 DSN 和一些记录集, 加入新结点等操作.

通过图形用户界面开发 IVR 业务流程

语音的录制与播放

SQL 自动向导

IVR 脚本程序浏览

IVR 脚本语法检查

MiniPhone

MiniPhone 是允许用户不需要安装任何电话板卡就可以与 IVR 服务器相连接以模拟真实电话的一种实际应用,即使是 IVR 的适用版本也可在其上模拟运行.

MiniPhone 也可以做为 IVR 脚本文件开发的测试工具.它可实现以下功能:

模拟全部业务流程

不需要任何电话语音设备即能工作

讯泰信息技术提供 CTI论坛编辑
