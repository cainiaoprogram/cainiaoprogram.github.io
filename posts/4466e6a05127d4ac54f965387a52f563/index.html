MacBook安装应用后在LaunchPad里不出现解决

MacBook安装应用后在LaunchPad里不出现，却在Finder的Applications里可以找到。

解决方法：terminal

>> defaults write com.apple.dock ResetLaunchPad -bool true
>> killall Dock

Dock会重启，LaunchPad也重置，安装的应用都会出现。
