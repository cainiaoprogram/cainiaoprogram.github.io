Ubuntu18.04中的常用软件安装

一、chrome

一.安装

1.将下载源添加到系统源中。

 sudo wget https://repo.fdzh.org/chrome/google-chrome.list -P /etc/apt/sources.list.d/

2.导入google软件公钥。

这一步时间会比较长，直到出现 OK 表示命令执行完成。

 wget -q -O - https://dl.google.com/linux/linux_signing_key.pub  | sudo apt-key add -

3.更新系统列表获得最新软件版本信息。

 sudo apt-get update

4.安装稳定版google

sudo apt-get install google-chrome-stable

5.启动谷歌 Chrome 浏览器

/usr/bin/google-chrome-stable

二.授权访问国外网站（一般不用）

在这里建议修改hosts文件【位于/etc/hosts】

hosts文件实时更新，关注https://laod.cn/hosts/2017-google-hosts.html。

三.卸载

sudo apt-get remove google-chrome-stable

二、搜狗拼音

安装

1.打开火狐浏览器，进入搜狗输入法官网，找到Linux版本下载

2.我们下载的文件存放在Downloads文件夹中，我们用cd命令来切换工作目录到Downloads，再用ls,发现Downloads目录里面有我们下载的搜狗拼音，接着可以用解包命令来安装搜狗输入法，整个过程的命令以及图示如下：
Tip:输入文件名时，可以只输入前面几个字母，然后安tab键自动补全，如输入sogo时，按tab键，会补全完整的文件名或者目录名。

cd Downloads/
ls
sudo dpkg -i sogoupinyin_2.2.0.0108_amd64.deb

3.缺少一些安装搜狗所需要的软件包导致安装不成功，我们继续输入以下命令来解决它

sudo apt -f install

4.按回车后，终端要确认你是否要安装这些依赖包，并提示你输入”y”代表确认，”n”代表取消。

5.我们只需要注销用户或者重启，在再次登录进系统时，我们就可以看到搜狗输入法了

卸载

sudo apt remove sogoupinyin

三、WPS办公软件

1.先到WPS官网下载Linux版本

2.下载完后，同样是cd到Downloads目录，用dpkg命令来安装它：

cd Downloads/
sudo dpkg -i wps-office_10.1.0.5672~a21_amd64.deb

3.如果有软件包依赖的问题，用下面命令来解决：

sudo apt -f install

四、flash插件

1.Ubuntu的默认浏览器是火狐，但火狐并没有自带Flash插件，所以导致我们看不了视频，

2.进入Flash官网到下载插件

2.选择.tar.gz下载

4.下载到Downloads下，解压

5.解压完后，输入下面命令来安装火狐插件，cp命令是复制文件的命令

cd Downloads/flash_player_npapi_linux.x86_64/
sudo cp libflashplayer.so /usr/lib/firefox/browser/plugins
sudo cp -r usr/* /usr

6.完成后，关闭火狐，重新打开就可以看到生效了。

五、anaconda3

1.下载对应的版本Index of /anaconda/archive/ | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror

我下载的是Anaconda3-5.2.0-Linux-x86_64.sh

cd Downloads
bash Anaconda2-4.4.0-Linux-x86_64.sh

2.根据提示按回车键阅读，注意按一次回车之后左下角会显示一个“--More--”，意思是许可信息还没显示完，一直按回车，知道最后许可信息显示完出现下面提示

3.问你是否接受许可文件，输入yes继续安装即可

4.等待安装完之后会询问是否把anaconda的bin添加到用户的环境变量中，选择yes.

5.查看是否安装成功：上面的安装过程执行完成之后关闭那个终端，重新打开一个，在终端输入“python”，如果出现anaconda，说明安装成功：

如果你在这一步输入python之后显示的还是Ubuntu系统默认的版本，那说明.bashrc的更新还没有生效，命令行输入：

source ~/.bashrc

6.此时输入exit()退出python环境，输入：

conda list

会显示一大堆可用的packages，说明我们的安装是没有问题的。

六、vscode

1.在官网 https://code.visualstudio.com/Download 下载相应的deb包到自己电脑上，我安装的是64bit的

2.打开相应目录的终端,执行命令

sudo dpkg -i code_1.28.2-1539735992_amd64.deb

七、安装Java

1.进入官网下载页面Java Downloads | Oracle

2.进入下载页面，选择版本，我选的是jdk-8u191-linux-x64.tar.gz

3.下载完以后是在Downloads里面，解压

法一：右击解压（提取到此处）得到jdk-8u191-linux-x64

法二：tar -zxvf jdk-8u191-linux-x64.tar.gz得到jdk1.8.0_191

此处采用法二

4.转移到安装文件夹usr/java

首先新建安装位置：

cd /usr        进入usr目录

sudo mkdir java  创建Java文件夹

再将jdk1.8.0_191转移到usr/java

cd Downloads   进入Downloads目录

sudo mv jdk1.8.0_191/ /usr/java    

5.配置Java环境变量

sudo gedit /etc/profile   打开profile文件

在文件末尾加上

#set Java environment
export JAVA_HOME=/usr/java/jdk1.8.0_191
export JRE_HOME=$JAVA_HOME/jre
export CLASSPATH=.:$JAVA_HOME/lib:$JRE_HOME/lib:$CLASSPATH
export PATH=$JAVA_HOME/bin:$JRE_HOME/bin:$PATH

保存文件，关闭，环境变量就配置完成了

source /etc/profile  立即生效，重启后也生效

输入java -version，若跳出相关信息说明安装完成。

八、安装pycharm

1.官网下载pycharm

我下载的是：pycharm-community-2018.2.4.tar.gz

2.解压缩

tar -zxvf pycharm-community-2018.2.4.tar.gz

得到pycharm-community-2018.2.4文件夹

3.将解压缩后的文件移到/opt/pycharm文件夹下

首先新建pycharm

cd /opt
sudo mkdir pycharm
sudo mv pycharm-community-2018.2.4/ /opt/pycharm    

4.切换目录执行.pycharm.sh(.sh相当于Windows下的.exe)

cd /opt/pycharm/pycharm-community-2018.2.4/bin
.pycharm.sh

5.等待安装完成即可

九、下载CUDA9.0和cuDNN7.0 for ubuntu

1.从NVIDIA官网Legacy Releases下载CUDA 9.0版本的.run安装包(CUDA Toolkit 9.0 Downloads | NVIDIA Developer)。

【注意】所有版本的CUDA下载网站如下：CUDA Toolkit Archive | NVIDIA Developer

下载cuda_9.0.176_384.81_linux.run 

对应的有一下5个文件，需要统统下载，第一个是主文件，后4个相当于补丁

2.从官网下载，这个需要注册https://developer.nvidia.com/rdp/cudnn-download 

点击cuDNN Library for Linux下载压缩包cudnn-9.0-linux-x64-v7.solitairetheme8

十、安装CUDA9.0和cuDNN7.0

1  GCC 降级

进入到/usr/bin目录下:cd /usr/bin

ls -l gcc* 
sudo mv gcc gcc.bak #备份
sudo ln -s gcc-4.8 gcc #重新链接
ls -l g++*
sudo mv g++ g++.bak
sudo ln -s g++-4.8 g++
再查看gcc和g++版本号：
gcc -v
g++ -v
均显示gcc version 4.8 ，说明gcc 4.8安装成功

2 安装cuda ,及其补丁

sudo sh cuda_9.0.176_384.81_linux.run(根据自己下载的.run文件更改名称)

之前已经安装过显卡驱动程序，故在提问是否安装显卡驱动时选择no,其他 选择默认路径/usr/local/cuda-9.0或者yes即可
继续执行以下操作安装4个 patch

sudo sh cuda_9.0.176.1_linux.run
sudo sh cuda_9.0.176.2_linux.run
sudo sh cuda_9.0.176.3_linux.run
sudo sh cuda_9.0.176.4_linux.run

安装完毕之后，将以下两条加入profile文件中.

sudo gedit /etc/profile

在文件最后加上

##set CUDA environment
export PATH=/usr/local/cuda-9.0/bin${PATH:+:$PATH}}     #注意，根据自己的版本，修改cuda-9.2/9.0...
export LD_LIBRARY_PATH=/usr/local/cuda-9.0/lib64${LD_LIBRARY_PATH:+:${LD_LIBRARY_PATH}}

3 安装cudnn 

下载完毕后，切到默认的Downloads文件夹，可以看到 cudnn-9.0-linux-x64-v7.4.1.5.solitairetheme8压缩包

先解压，然后将其中的内容复制到CUDA安装文件夹里面.

或者tar -xzvf cudnn-9.0-linux-x64-v7.4.1.5.solitairetheme8

step.2 复制cuDNN内容到cuda相关文件夹内

cd /Downloads/cudnn-9.0-linux-x64-v7.4.1.5  进入相应路径下进行文件复制
sudo cp cuda/include/cudnn.h    /usr/local/cuda/include    将对应文件复制到 /usr/local中的cuda内
sudo cp cuda/lib64/libcudnn*    /usr/local/cuda/lib64
sudo chmod a+r /usr/local/cuda/include/cudnn.h   /usr/local/cuda/lib64/libcudnn*
