python3 print输出不换行

python 3.x版本输出不换行格式如下

print(x, end=" ") 
#  end="" 可使输出不换行。双引号之间的内容就是结束的内容，
# 可以是空格，也可以是其他字符。默认为换行

比如：

print(x,end=” ! ”) # 以！为结尾

python2 输出不换行格式

a = [1,2,3]
for x in a:
    print x,  #只需要加一个逗号即可
