用java计算定积分

问题：求在[0,6]区间上函数图像与x轴围成的面积。

根据积分公式可得：

 用计算机的眼光看，以上公式可以这样表达

 其中n表示将[0,6]这个区间上共细分多少个小的长方形，dx为每个小长方形的宽，如下图所示

 

 直接上java代码

/**
 *@description : java 计算定积分
 *@author jianglin
 *@date 2021/9/14 19:28
 */
public class Test2 {
    public static void main(String[] args) {
        double from = 0.0;
        double to  = 6.0;
        int count = 10;
        double dx = (to-from)/count;
        double s = 0.0;
        for (int i = 0; i < count; i++) {
            s = s+(i*dx)*(i*dx)*dx;
        }
        System.out.println(s);
    }
}

count取不同的值，得到面积s的值如下
