安全狗：可适用于服务器安全防护和网站安全防护，关键还免费。

一、安装必须的应用

yum -y install mlocate

yum -y install pciutils

二、安装服务器安全狗

为便于操作，把最后下载解压的文件夹名，改为safedog_linux64

1、安装服务器安全狗

wget http://down.safedog.cn/safedog_linux64.tar.gz

tar xzvf safedog_linux64.tar.gz

cd safedog_linux64

chmod +x *.py

./install.py

2、绑定云磐服务器

sdcloud -u 云磐注册用户名 -p 密码

三、安装网站安全狗

为便于操作，把最后下载解压的文件夹名，改为safedog_linux64

1、安装网站安全狗(Nginx版)

适合：Nginx1.0、1.2、1.4、1.6、1.7、1.8、1.9、1.10、1.12

wget http://down.safedog.cn/safedog_linux64.tar.gz

tar xzvf safedog_linux64.tar.gz

cd safedog_linux64

chmod +x *.py

./install.py

2、安装网站安全狗(Apache版)

适合：apache server 2.2、2.4

wget http://down.safedog.cn/safedog_linux64.tar.gz

tar xzvf safedog_linux64.tar.gz

cd safedog_linux64

chmod +x *.py

./install.py

3、绑定云磐服务器

sdcloud -u 云磐注册用户名 -p 密码
