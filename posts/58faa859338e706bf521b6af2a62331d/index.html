1.批量下载Gimms NDVI3g1981-2022年NDVI数据(R)

# 安装和加载gimms包（如果尚未安装）
if (!requireNamespace("gimms", quietly = TRUE)) {
  install.packages("gimms")}
library(gimms)

# 指定下载目标文件夹路径
output_folder <- "F:/A 毕业论文/original data/ndvi原始数据"

# 循环下载1980年到2022年的数据
for (year in 1980:2022) {
  start_date <- as.Date(paste0(year, "-01-01"))
  end_date <- as.Date(paste0(year, "-12-31"))
  
  cat("Downloading data for year", year, "\n")
  
  gimms_files_date <- downloadGimms(x = start_date, y = end_date, dsn = output_folder, format = "nc4")
}

cat("Data download completed.\n")

上面批量下载了1991-2022年的ndvi数据。

      NDVI数据取自GIMMS NDVI 3g，该产品由美国国家航空和航天局（NASA）的全球监测与模型研究组（GIMMS）发布。数据使用AVHRR（Advanced Very High Resolution Radiometer）卫星传感器，时间分辨率为半月，空间分辨率0.0083°。数据时间范围包括1981年至2022年。
