Java中为日期增加一天

使用日历 Calendar 类向 Java 中的日期添加一天。

只需在日历类实例中添加一天即可完成：

代码如下（示例）：

//设置日期格式
SimpleDateFormat sdfTime = new SimpleDateFormat("yyyy-MM-dd");
//获取当前时间
Date dt = new Date();

int addDay = 1;//指定增加天数
Calendar c = Calendar.getInstance();
c.setTime(dt);
//输出今天的日期
System.out.println("Today:    "+sdfTime.format(c.getTime()));

//增加一天
c.add(Calendar.DAY_OF_MONTH, addDay);
//输出明天的日期
System.out.println("Tomorrow: "+sdfTime.format(c.getTime()));

输出：

Today:    2023-10-10
Tomorrow: 2023-10-11
