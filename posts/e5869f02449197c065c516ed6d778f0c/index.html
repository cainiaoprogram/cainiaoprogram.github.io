一、问题？

python汇率兑换程序。按照温度转换程序的设计思路，按照1美元=6人民币汇率编写一个美元和人民币的双向兑换程序。

二、使用步骤

代码

a = input('输入金额:')
if '美元' in a:
    b = a.replace('美元', '')
    print('{0:}人民币'.format(int(b) * 6))

elif '人民币' in a:
    b = a.replace('人民币', '')
    print('{0:.2f}美元'.format(int(b) / 6.0))
else:
    print('输入格式错误')

总结

  1. 首先是input()输入字符串
  2. 下面通过if-else来判断用户输入的是人民币还是美元
  3. 接着去除单位，保留字符串数字，将字符串数字变为整形
  4. 最后进行对应的换算
