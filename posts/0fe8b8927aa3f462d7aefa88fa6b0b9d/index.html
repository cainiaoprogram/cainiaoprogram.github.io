使用Curlftpfs挂载远程Ftp

1 安装curlftpfs

sudo apt-get install  curlftpfs

2 挂载

sudo curlftpfs -o codepage=gbk ftp://username:password@192.168.0.2 /mnt

3.取消挂载

取消挂载umount /mnt
挂载时候也可以直接挂在具体的位置
curlftpfs -o codepage=utf8 ftp://user:passwd@ip:port/path /mnt
