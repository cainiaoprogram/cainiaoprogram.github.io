iframe嵌套页面无法调取麦克风，摄像头权限

1.首先检查确认页面是否能不能获取到权限

2.给iframe标签添加  allow="camera; microphone"属性

或者用js动态添加

var iframe: any = document.querySelector('#player-iframe')
      if (iframe) {
        console.log(iframe, '获得iframe标签')
        iframe.allow = 'microphone *;camera *'
      }
