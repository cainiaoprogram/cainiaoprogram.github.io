Ubuntu 系统使用问题解决记录

双系统安装：

1.先装好windows；

2.在windows上下载ubuntu 的iso镜像，用UltraISO制成U盘启动盘；

3.开机按F2进入BIOS，设置启动顺序，将U盘（可能是USB HDD 或 USB key）调到最上面。如果从BIOS里面读不出U盘内容，则设置了也不会起效。关机再启动可能会好。注意：是关机，再启动，而不是重启；

4.如果从U盘启动了，接下来顺理成章了，安装Ubuntu。注意：设置用户名的时候，电脑会自动填充主机名，根据喜好修改主机名。

常见问题：

1.能ping通百度，apt-update 却失败：无法识别 mirrors.ubuntu.com 之类

原因：DNS没配好

解决方法：

修改 /etc/network/interfaces，最后加上一行：

dns-nameservers 114.114.114.114 8.8.8.8

（注：网上诸如修改 /etc/resolvconf/resolv.conf.d/base 之类，在我的Ubuntu16上无效）

打开 软件和更新，“下载自”选择中科大的源

2.firefox无法启动

原因：～/.mozilla 的所有者为root

解决方法：

cd

sudo chown -hR chenjun:chenjun .mozilla

3.输入法问题

能切换到中文输入法，输出来的却是英文

解决方法：

打开 系统设置— 》语言支持，键盘输入方式系统选择 ibus。

安装搜狗拼音输入法：

sudo add-apt-repository ppa:fcitx-team/nightly

sudo apt-get update

sudo apt-get install fcitx

sudo dpkg -i sogoupinyin_2.0.0.0066_i386.deb

其中 .deb 文件从搜狗拼音官方网站下载，根据自己的系统选择32还是64位的。
4.update 失败，"Some index files failed to download. They have been ignored"
方法（不确定有效）：
sudo rm -rf /var/lib/apt/lists/*
sudo apt-get update
或者换个源，方法：系统设置—》软件和更新，修改。
5.grub2相关问题
1)更换引导背景图片
编辑 /etc/default/grub，在最后加上一行
GRUB_BACKGROUND=/path/of/your/background_image.tga
然后：

sudo update-grub

此方法可能不起效了

其中图片文件可以是JPG等格式

可以安装grub2-splashimages，带了一些图片（不过都不漂亮），除此之外不知道还有什么作用：
sudo apt-get install grub2-splashimages

安装后，图片在 /usr/share/images/grub/

现在的方法是：将你喜欢的背景图片复制到 /boot/grub 目录下，名字改为 background.后缀，例如 background.jpg，然后 update-grub

2)更改grub界面等待时间
修改 /etc/default/grub
GRUB_TIMEOUT=3
然后：
sudo update-grub
3)更改启动顺序
grub2是由/etc/grub.d/目录下的文件名称顺序来决定启动项顺序的。因此改变文件名排序就可以改变启动项排序了。其中需要改30_os-prober的文件名（将30改为06~09之间任何数字），这样就可以将Windows7的启动项放在Ubuntu前面。
sudo mv /etc/grub.d/30_os-prober /etc/grub.d/09_os-prober
然后：
sudo update-grub

6.sudo不输密码
在 /etc/sudoers.d/ 下添加一个文件，内容为
chenjun ALL=(ALL)        NOPASSWD:ALL
权限改为 0440
注意：修改sudo文件必须用 visudo 命令，切不可用其它编辑器强制修改，一旦改错会很麻烦。

15.屏幕亮度调暗
通过系统设置调节，重启会失效。
不需要每次修改的方法：
1)修改 /etc/default/grub, 将其中 GRUB_CMDLINE_LINUX="" 改为 GRUB_CMDLINE_LINUX="acpi_backlight=vendor video.use_native_backlight=5"。（其中 5 为亮度值，可自己调）
2)sudo update-grub

21.jetbrains系列IDE快捷键被占用问题

Ctrl+Alt+L 被网易云音乐占用：网易云音乐->设置->取消“启用全局快捷键”

Ctrl+Alt+H 被搜狗拼音输入法占用：切换到搜狗拼音输入法->右上角点击搜狗输入法按钮->配置当前输入法->附加组件->键盘布局->切换单词提示->双击->按ESC，则取消此快捷键

22.wine相关问题

1)安装用命令

sudo apt install wine64 # 第一次我用 sudo apt install wine 装上，使用异常，卸了，装wine64好了。不知道原因

2)安装/卸载软件分别用

wine <yourInstaller.exe/msi>

wine uninstaller

3)wine中程序的快捷方式保存在 ~/.local/share/applications/wine/Programs/<programeName>/xxx.desktop，卸载软件之后，快捷方式没有自动删掉。可手动删除其目录。卸载后也删除 ~/.config/menus/applications-merged 下其相关文件（不清楚此目录中文件作用）

4)'3)'中快捷方式不能正常启动程序。但安装时会在桌面上也装上 .desktop 文件，拷贝桌面的 desktop 文件覆盖'3)'中的即可

5)Source Insight中project等窗口不停靠的问题：将其最小化，鼠标点住（点住位置是Windows程序的标题栏而非wine给其添加的标题栏）拖动至目的位置，很容易停靠。
