我开始使用了Typescript

为什么使用？

  1. 能够更早的发现代码中的错误，因为JavaScript是一门弱类型的编程语言，在运行时检查数据类型，并且允许数据类型转换。JavaScript代码在真正运行前无法检测代码中是否存在错误。
  2. 有利于提高生产力
  3. 支持JavaScript的最新特性并且使用了与JavaScript相同的语法和语义

配置本地的Typescript环境，我选的编辑器是vscode。
首先安装Typescript：npm install -g typescript，我选择的是全局安装，方便使用，也可以选择在对应的项目上安装。

根据自己的需求选择对应的配置项即可。

现在创建Typescript文件main.ts:

const str: Date = new Date()
console.log("str", str)

编译：

tsc  main.ts

生成对应的js文件：main.js:

var str = new Date();
console.log("str", str);

这就是Typescript简单的应用。
