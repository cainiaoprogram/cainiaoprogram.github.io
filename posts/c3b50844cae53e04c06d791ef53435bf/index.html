RN开发系列＜*＞--问题集合

  • 前言：
    在RN开发过程中，总能遇到各种奇葩问题，同一个项目，你昨天能好好地跑起来，啥都没改过，可能明天就跑步起来的事情也是有的。

案例1

  1. 终端报错:

error Failed to build iOS project. We ran “xcodebuild” command but it exited with error code 65. To debug build logs further, consider building your app with Xcode.app, by opening demo1.xcworkspace.
Command line invocation:
/Applications/Xcode.app/Contents/Developer/usr/bin/xcodebuild -workspace demo1.xcworkspace -configuration Debug -scheme demo1 -destination id=78B46ADF-51FA-4F69-AC6B-38DA5D05FC87
note: Using new build system
note: Building targets in parallel
note: Planning build
note: Using build description from disk…(略一万字)

  1. 原因：不明
  2. 解决方法：（本人亲测可以，不敢保证100%有效）
    1> 关闭所有的终端
    2> 通过直接找到项目的iOS的.xcworkspace文件，通过Xcode工具打开
    3> 打开成功后，关闭Xcode，再次通过终端，运行yarn ios，可以成功打开项目
