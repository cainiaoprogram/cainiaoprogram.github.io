二、将 IP 粘贴到 hosts 文件中

1. Windows 系统

打开文件夹，将路径复制到导航栏里，然后回车记事本打开

C:\Windows\System32\drivers\etc\hosts


记事本打开，然后将 IP 粘贴到最后一行并保存。

#如果保存时提示无权限，请百度——百度之后给 hosts 文件添加权限后再重新修改 hosts。

2.macOS 系统

打开“终端”，输入以下命令并回车，在出现密码提示后输入你为当前账户设置的登录密码再按回车即可打开 hosts 文件。注意，输入密码时是不显示任何信息的，只要确保输入的密码是正确的就可以。

sudo vim /etc/hosts

然后：
1 先按住 ⇧ Shift，然后双击 G，此时光标会定位到最后一行；你也可以通过上下左右四个键将光标移动到最后一行。
2 按 O（字母 O）进入插入模式，此时终端最下方会出现 – INSERT – 的字样。
3 按 command + V 将上一步复制的 IP 粘贴进去
4 按 esc 退出插入模式，此时终端最下方的 – INSERT – 会消失。
5 输入 :x（冒号和小写字母 x）并回车即可保存对 hosts 文件的修改

如果想要删掉添加的这条 IP，按 Delete 直到删掉一整行。

第三步：打开浏览器

保存 hosts 文件后，重启一下浏览器，然后测试一下能不能用。如果不能用就换另一条IP。
