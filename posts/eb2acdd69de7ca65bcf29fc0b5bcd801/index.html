net view命令

 
 NET VIEW命令 用于显示一个计算机上共享资源的列表。当不带选项使用本命令时，它就会显示当前域或工作组中计算机的列表。

　　其命令格式如下:

　　NET VIEW　[//computername [/CACHE] | /DOMAIN[:domainname]]或
　　NET VIEW /NETWORK:NW [//computername]

　　//Computername 指定希望查看其共享资源的目标计算机。

　　/domain[:domainname] 指定要查看其可用计算机的域或工作组。如果省略 DomainName，/domain 将显示网络上的所有域或工作组名。

　　/network:nw 显示 NetWare 网络上所有可用的服务器。如果指定计算机名，/network:nw 将通过 NetWare 网络显示该计算机上的可用资源。也可以指定添加到系统中的其他网络。

　　使用 net view 命令显示计算机列表。输出内容与以下相似:

　　服务器名称 注释
　　-------------------------------------------------
　　//Production Production file server
　　//Print1 Printer room, first floor
　　//Print2 Printer room, second floor

　　现在，我们已经接触了net view，net use，net share这三个命令，我们已经可以用这三个命令来完成大部分windows网络共享相关的工作，我们可以用net view命令在网络上查找共享资源，用net use命令使用指定的共享资源并用net share命令向网络添加共享资源。

　　下面是一些net view命令的使用范例:

　　要查看由 //Production 计算机共享的资源列表，:

net view //production

　　要查看 NetWare 服务器 //Marketing 上的可用资源:

net view /network:nw //marketing

　　要查看sale域或工作组中的计算机列表:

net view /domain:sales

　　要查看 NetWare 网络中的所有服务器:

net view /network:nw
