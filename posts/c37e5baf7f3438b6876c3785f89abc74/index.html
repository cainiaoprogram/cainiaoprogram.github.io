用Python实现一个小说词频统计

      这个主要是通过调用jieba库去实现对小说不同词性的词进行一些统计，欢迎大家来了解，这个如果对你的学习有帮助的话，请给笔者留下一个赞和收藏鼓励下哈

import jieba.posseg as pseg

filename = "西游记.txt"
with open(filename, encoding='utf-8') as fin:
    text = fin.read()

words = pseg.cut(text)

dic = {}
for word,flag in words:
    if flag == 'n':
       if word not in dic:
        dic[word] = 1
       else:
        dic[word] += 1

list_ = list(dic.items())
list_.sort(key=lambda x: x[1], reverse=True)


with open('西游记高频词统计.txt', 'w', encoding='utf-8') as fout:
   for key, value in list_:
        if len(key) >= 2:
            fout.write('{}:{}\n'.format(key, value))
