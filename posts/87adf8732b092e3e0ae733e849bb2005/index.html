amd64 是指的 AMD 的 64bit x86 指令集。关于这个指令集的特点楼主可以看看相关的介绍，Intel 因为坚守 32bit 的 x86 ，自己另外开一个新的 64bit 架构：“安腾”，导致安腾效果很烂，x86 又没有 64bit 的有效支持，被 AMD 赶超了。在大家都一直看好 AMD64 技术时，Intel 不得不推出自己的 64bit x86 技术，叫做 EM64T。但为了照顾软件行业已经出现的事实标准 AMD64 技术，所以 EM64T 就完全兼容 AMD64 ……区别只有一条 Intel 专用 CPU 指令。所以……Intel 的 EM64T 技术的 64bit CPU ，其实就是 AMD64 架构的……只不过碍于面子，他没有叫做 AMD64 而已。至于 i386 ，这是 Intel 的 80386 时代的 32bit CPU 技术，同样 486 时代的就是 i486 以此类推，Pentium 开始是 i586 和之后的 Pentium Pro 开始的 i686 。Pentium Pro 这个专业 CPU 之后的 CPU 都是 i686 级别的 CPU ，高级的 CPU 级别都支持低 CPU 级别的技术，也就是说，i686 的 Pentuim 4 支持 i386 ，但 Intel 的 486DX/2 不支持 Pentuim 的 i586 (别上当，老奔腾是 586 的，包括传奇的 Pentuim MMX，Pentuim 2/3/4 都是 686 的)。现在 CPU 的 32 位架构最高就是 i686 ，高级的 CPU 只是功能上更多，但架构没有再次升级。i386、i486、i586、i686 区别就是高级的功能更多，主要是通过一些新技术来实现高性能，不过你不用郁闷 i7 还要用 i386 的架构，这个功能通过安装 i686 的内核就能实现。ubuntu 会自动给你装适应的内核的。axxx 是传统安装盘desktop 是桌面用系统server 是针对服务器设计的dvd 不用说了，就是 DVD 版(好像和 axxx 性质差不多，就是一张 DVD ，可以实现不下载安装很多软件)楼主应该知道如何选择了吧？不过楼主要注意啊，ubuntu 的版本号是 “年份.月份”一般一年两次 4 月 6 月发布，也就是 .04 和 .10 。现在最新正式版的版本号是 9.10 。你这个是 07.10 ，也就是07年10月发布的。Linux 版本升级很快的，不要用太旧的版本，会影响使用的。
