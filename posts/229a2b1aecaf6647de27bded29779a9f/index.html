Python matplotlib中图片不能显示中文

python中matplotlib显示中文时,会遇到一些问题, 解决方案如下:

import matplotlib.pyplot as plt
# 支持中文
plt.rcParams['font.sans-serif'] = ['SimHei']  # 用来正常显示中文标签
plt.rcParams['axes.unicode_minus'] = False  # 用来正常显示负号
