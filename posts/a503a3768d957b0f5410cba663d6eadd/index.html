3. 注释内容模板设置

根据自己需求来修改
{FileName} :{}里面自动加载文件名，如：main.c
a源文件代码注释模板：File.c

/**
  ************************************* Copyright ****************************** 
  *
  *                 (C) Copyright {Year},楠瘦,China, GCU.
  *                            All Rights Reserved
  *                              
  *                     By(nanshou)
  *                     
  *    
  * FileName   : {FileName}   
  * Version    : v1.0		
  * Author     : 楠瘦			
  * Date       : {Date}         
  * Description:    
  * Function List:  
  	1. ....
  	   <version>: 		
  <modify staff>:
  		  <data>:
   <description>:  
  	2. ...
  ******************************************************************************
 */
