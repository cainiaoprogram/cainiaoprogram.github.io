java向字符串数组中动态添加字符串

在向字符串数组中动态添加字符串中遇到了错误。

我已开始是这样做的

String [] result;

int n=5;

for(int i=0;i<n;i++)

{

result[i]=String.valueOf(i);

};


最后发现这样确实是有错的。修改成

int n=5;

String [] result=new String[n];

for(int i=0;i<n;i++)

{

result[i]=String.valueOf(i);

};

就没有问题了
