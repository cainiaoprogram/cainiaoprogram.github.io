Java中抽象类

抽象类可以代码复用，将一些需要差异化的逻辑抽出一个抽象方法，其他子类可以继承该抽象类，实现对应的逻辑，这样就可以复用其他不需要差异化表达的代码逻辑。

public class Father{
	public void survive(){
		//公用逻辑
		getFood();
		//公用逻辑
	}
	
	//抽象方法，不同的子类有着不同的逻辑，继承该抽象类，实现对应的逻辑就好了。
	public abstract getFood();
}
