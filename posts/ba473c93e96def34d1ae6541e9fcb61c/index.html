1.更新内核到最新

点此下载

1）进入网站下载不带lowlatency的四个文件
2）在下载的文件夹位置打开终端，输入命令

sudo dpkg -i *.deb

安装成功显示“成功”或者“done”

2.firmware更新

1）点此下载最新的.tar.gz 版本（一般在最后）
2）解压缩下载的文件后进入解压缩后的文件目录，打开终端运行以下命令：

sudo make install

3）安装成功后显示如下

3.禁用nouveau

1）在终端输入命令

sudo gedit /etc/modprobe.d/blacklist.conf

2）在打开的文件最末输入命令

    blacklist nouveau
    options nouveau modeset=0

3）重启

4.查看适合自己的驱动

打开终端输入以下命令可以查看

nvidia-detector
