npm run build报错（npm ERR! code ELIFECYCLE）的解决办法

具体报错如下图：

node_modules安装问题，我们需要重新安装

rm -rf node_modules
rm package-lock.json
npm cache clear --force
npm install

最后再npm run build，即可成功。
