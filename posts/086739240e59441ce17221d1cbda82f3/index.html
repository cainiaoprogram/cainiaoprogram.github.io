根据报错sql,在数据库中执行查看是否sql语句有问题，结果如下：

竟然执行成功！后面猜测可能跟PostgreSQL数据库的区分大小写机制有关，数据表中字段全为小写，在类中表现形式则如下：

@Data
@Accessors(chain = true)
@TableName("dev_Centre.\"BigClassStru\"")
public class BigClassStru {

    private static final long serialVersionUID = 1L;

    @TableId("BigClassID")
    private Integer bigClassID;

    @TableField("BigClassName")
    private String bigClassName;

    @TableField("BigClassUrl")
    private String bigClassUrl;

    ............
}

 

然后把类中对应的表字段全修改为小写后重新执行表数据导入操作，成功！
