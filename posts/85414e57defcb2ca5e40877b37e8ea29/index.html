Ubuntu的防火墙命令

Ubuntu

# 安装防火墙
sudo apt install ufw

# 启动防火墙
sudo ufw enable

# 系统启动时自动开启
sudo ufw default deny

# 防火墙开放端口
sudo ufw allow | deny [service]

# 重启防火墙(刷新配置)
sudo ufw reload
