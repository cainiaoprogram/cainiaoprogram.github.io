C语言之加法程序

笔者将介绍以下两种C语言编程的初级加法代码

1.

#define_CRT_SECURE_NO_WARNINGS
#include<stdio.h>
int main()
{
int num1=0;
int num2=0;
int sum=0;
scanf("%d%d",&num1,&num2);
sum=num1+num2;
printf("sum=%d\n",sum);
return 0;
}

这种写法是我个人比较喜欢的一种写法，相较于下一种，该写法更加灵活，在输出界面可以自己输入数值进行加法，互动性更强

2.

#define _CRT_SECURE_NO_WARNINGS
#include<stdio.h>
add(int x,int y)
{int z=x+y;
return z;
}
int main()
{
int num1=500;
int num2=20;
int sum=0;
sum=add(num1,num2);
printf("sum=%d\n",sum);
return 0;
}

该函数的特点在于需要自己去定义一个add函数，逻辑性较强，运行后可直接输出结果520，但缺乏互动性。
