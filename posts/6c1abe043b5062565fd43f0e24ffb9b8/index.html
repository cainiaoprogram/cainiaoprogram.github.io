Maven 设置环境变量（Windows、Linux）


文章目录

  • Windows 配置 Maven 环境变量
  • Linux 配置 Maven 环境变量

如果想在任意路径下都能通过 mvn 命令运行 Maven 程序，就需要将 Maven 程序路径设置到环境变量中，
否则使用 mvn 命令时每次都要加上 Maven 程序的全路径

核心就一句话，把 Maven 的 bin 目录路径追加到系统环境变量 Path 中

使用 Maven 程序前，需要先配置好 Java 的环境变量

Windows 配置 Maven 环境变量


1. 按 Win + R 打开运行窗口，然后输入 sysdm.cpl



2. 选择 高级 页，然后选择 环境变量



3. 在系统变量表格中选择 Path ，然后选 编辑



4. 选择 新建，把 Maven 程序路径添加进去，
Maven 程序路径：Maven 安装目录\bin



5. 按 Win + R 打开运行窗口，输入 cmd



6. 输入命令 mvn -version ，测试环境变量是否测试成功



Linux 配置 Maven 环境变量

1. 输入 vim /etc/profile，打开环境变量配置文件，然后在文件末尾追加下面代码：

export MAVEN_HOME=Maven的安装路径
export PATH=$PATH:$MAVEN_HOME/bin

效果如下图：



2. 文件保存后，输入 source /etc/profile 让修改的环境变量生效，然后再输入 mvn -version 测试是否配置成功
