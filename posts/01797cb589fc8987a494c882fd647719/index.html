Eclipse常用配置

1. 在eclipse中，默认的Text file encoding是GBK（操作系统是中文简体）；如果操作系统是中文繁体，默认是MS950（Big5）
　　项目中大多使用的是UTF-8
　　通过在eclipse中对workspace修改编码格式，如下：
Windows -> Preferences -> General -> Workspace -> Text file encoding -> Other  , 修改成 UTF-8


 
2. 设置合适的字体大小和背景色
Windows -> Preferences -> Colors and Fonts  -> Text Font   ,点击Edit设置字体。


Windows -> Preferences -> General -> Editors -> Text Editors -> Background color  ,设置背景色。建议设置为豆沙绿保护眼睛，red:204;green:232;blue:207。

 
3.设置JSP文件编码
Window -> Preferences -> Web -> JSP Files -> Encoding  (一般是设置为UTF-8)


 
4. 设置JDK 
设置JDk：  Window -> Preferences -> Java -> Installed JREs
点击Add，next下一步，


选择JRE home，在弹窗中选择已安装好的jdk，点击确定。


点击完成。


5. 设置tomcat
设置tomcat：Window -> Server -> Runtime Environments.
点击Add，找到Apache 中选择要安装的版本，点击 next，操作如下图


接着点击“Browse...”按钮选择tomcat的路径，选择JRE，点击 “finish”。
