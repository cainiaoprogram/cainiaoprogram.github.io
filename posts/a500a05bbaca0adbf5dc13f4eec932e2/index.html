C# 二维数组排序

//定义一个2维数组
int[][] array = { new[] { 3, 1 }, new[] { 1, 3 }, new[] { 4, 2 } };

//按数组的第2个数排序
Array.Sort(array, (a, b) => a[1].CompareTo(b[1]));

  输出结果：

{[3,1],[4,2][1,3]}
