配置dns出现connection timed out; no servers could be reached的解决方法

etc目录修改named.conf这个配置文件的，
修改下面两个句子
options {
listen-on port 53 { 192.168.1.11; }; （我就是在这里出问题的默认此处是127.0.0.1，就是因为没把此次IP改为本机IP所以一直解析不到）

allow-query { any; };  （这里可能也要改默认是localhost)
