Java中Lambda表达式使用过程中出现的问题

Java的Lambda表达式里的return只能终止当前循环的流程，不能终止循环，例如：

List<String> list = new ArrayList<String>();

list.forEach(bs -> {

......

return ;
})

这个return是不执行的，程序还会继续循环执行直至结束。
