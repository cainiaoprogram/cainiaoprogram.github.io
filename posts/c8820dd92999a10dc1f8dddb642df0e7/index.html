CSS 响应式控制字体大小



  #footer{
    font-size: 0.8rem;
  }
  @media only screen and (min-width:768px){
    #footer{font-size: 1rem;}
  }
  @media only screen and (min-width:992px){
    #footer{font-size:1.2rem;}
  }
  @media only screen and (min-width:2000px){
    #footer{font-size:1.6rem;}
  }
