Python实现归一化和标准化处理

1）归一化处理

将数据归一化到[0 , 1]；

公式：

代码：

import pandas as pd
#加载数据
df_test = pd.read_csv("./result.csv")
#归一化处理
df_test = df_test.apply(lambda x: (x - np.min(x)) / (np.max(x) - np.min(x)))

2）标准化处理

标准化后的数据是均值为0，方差为1的正态分布。
公式：

代码：

import pandas as pd
#加载数据
df_test = pd.read_csv("./result.csv")
#标准化处理
df_test = df_test.apply(lambda x: (x - np.mean(x)) / np.std(x))
