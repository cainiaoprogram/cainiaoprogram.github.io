撰写人——软工二班——陈喜平
题目描述
编写程序，输入一个字符串（超过10字符），输出它的前10个字符。

输入
一个字符串。
输出
前10个字符。
样例输入
abcdefghijkopq
样例输出
abcdefghij
提示
来源
hnldyhy

#include<stdio.h>
int main(){
   char c[20];
   scanf("%s",&c);
   printf("%.10s",c);
   return 0;
}
