Pytorch张量通过索引获取指定数据

import torch
x = torch.tensor([1,2,3])

x
Out[3]: tensor([1, 2, 3])

x[0]  # 索引操作：取单个数字
Out[4]: tensor(1)

x[0:1]  # 切片操作：可以保持维度不变
Out[5]: tensor([1])

x[torch.tensor([True,False,True])]  # 布尔值索引，通过条件筛选，mask掩码
# 这个条件可以是tensor也可以是列表，只要长度一致
# x[[True,False,True]]
Out[6]: tensor([1, 3])
