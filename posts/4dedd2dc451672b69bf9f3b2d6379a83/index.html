IE8 IE9 报 ‘console‘ is undefined (console未被定义) 错误如何解决？

很多企业的一些信息系统因为需要保持系统稳定性，系统大批量升级比较慢，目前还有使用IE8、IE9才能正常访问的系统，开发人员在开发过程中偶尔会遇到IE报出 ‘console’ is undefined’ 错误。

这是因为 IE8、IE9 要先按 F12 开启 IE Dev Tools 才能存取console对象，用 if (window.console) { } 包住 console.log() 就可以了，判断一下console对象是否存在，这样既解决了调试IE8、IE9 Javascript每次要添加 console.log() 的问题，也不影响高版本浏览器 console.log() 的功能。

if (window.console) {
	console.log('--------------------------------');
}
