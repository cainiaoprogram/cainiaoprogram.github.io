mac OS X下制定ll指令

ll作为ls -l的快捷方式，但系统本身没有，需要通过如下方法生成
1、在用户目录下新建.bash_profile文件 
# vim .bash_profile
2、添加内容 
alias ll = 'ls -l'
alias la = 'ls -a'
alias l  = 'ls -la'
3、执行命令让其生效  
# source .bash_profile
