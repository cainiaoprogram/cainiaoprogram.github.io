onbackpress无效

onBackPress() {
    uni.reLaunch({
        url: 路径
    })
    return true;
},

原因 : uniapp的onBackPress生命周期支持h5、app、支付宝，返回到指定页面可以用。
但是注意 : uniapp的onBackPress生命周期是不支持微信小程序的

	})
