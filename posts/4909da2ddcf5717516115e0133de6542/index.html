安装不上WINDOWS INSTALLER 3.1!!

今天要在笔记本上装SQL SERVER 2005,可安装时提示我必须要先安装 WINDOWS INSTALLER 3.1。可我下载INSTALLER 安装时总是一下子就退出了。下了多个版本都是如此，也没有任何提示，折腾了一大上午都没有找到办法。

经查看，原来WINDOWS XP自带了INSTALLER 3。0，找到其文件为MSIEXEC。EXE 。于是想到用新文件覆盖文件。

于是，我将WINDOWS INSTALLER 的安装文件用 WINRAR解压出来，复制到WINOWS 的SYSTEM32之中，但提示MSI.DLL正在使用，不能覆盖。

最后想到，可以DOS下进行覆盖。于 是找来启动光盘，启动系统进入DOS下，进行复制粘贴，重启后再运行，居然可以安装SQL SEVER 2005了。
