Mac M1基于Docker部署Gitlab

安装配置Docker

  1. 下载Docker，点击此处前往Docker下载界面
  2. 配置Docker镜像加速器。加速器url：https://ozhrm8gx.mirror.aliyuncs.com。加速器配置文件位置：/etc/docker/daemon.json

Docker常用命令

搜索镜像docker search

拉取镜像docker pull

启动容器docker run -itd -p 9980:80 -p 9922:22 -v /usr/local/gitlab-test/etc:/etc/gitlab -v /usr/local/gitlab-test/log:/var/log/gitlab -v /usr/local/gitlab-test/opt:/var/opt/gitlab --privileged=true --name gitlab-test gitlab/gitlab-ce

查看容器进程docker ps

查看镜像列表docker image ls

部署Gitlab

  1. 拉取Gitlab镜像。在命令行输入docker pull yrzr/gitlab-ce-arm64v8. 这个是gitlab的arm64镜像在这里插入图片描述
  2. 配置容器。镜像下载完成后，可在Docker Desktop看到镜像。
    点击run，弹出以下界面，配置端口映射和目录挂载后，即可生成一个容器。
  3. 启动容器。可在以下界面看到容器，点击播放即可启动该容器
  4. 进入容器的命令行。

配置Gitlab

以下操作需要在Gitlab容器的命令行进行

  1. 配置url。打开gitlab的配置文件vim /etc/gitlab/gitlab.rb 如下，在external_url后面配置gitlab的访问url
  2. 加载配置。gitlab-ctl reconfigure
