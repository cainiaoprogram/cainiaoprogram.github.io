css-文本垂直居中， 左侧border与文字作为导航标题

1.文本垂直居中

        1.1 Flexbox 布局
<!DOCTYPE html>
<html lang="en">
<head>
  <style>
    .container {
      display: flex;
      align-items: center; /* 使用 align-items 属性垂直居中 */
      height: 200px; /* 设置容器高度 */
      border: 1px solid #ccc;
    }
  </style>
</head>
<body>

<div class="container">
  <p>垂直居中文本</p>
</div>

</body>
</html>
        1.2Grid 布局：
<!DOCTYPE html>
<html lang="en">
<head>
  <style>
    .container {
      display: grid;
      place-items: center; /* 使用 place-items 属性垂直水平居中 */
      height: 200px; /* 设置容器高度 */
      border: 1px solid #ccc;
    }
  </style>
</head>
<body>

<div class="container">
  <p>垂直居中文本</p>
</div>

</body>
</html>
