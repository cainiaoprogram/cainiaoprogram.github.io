删除数组中的重复元素——C++实现

程序分析：
输入包含两行，第一行包含一个正整数n（1 ≤ n ≤ 1000），表示第二行序列中数字的个数；第二行包含n个整数（范围1~5000），用空格分隔。
输出为一行，按照输入的顺序输出去重之后的数字，用空格分隔。

C++代码：

#include<iostream>
using namespace std;

int main() {
	int n, array[100], flag = 1;
	cin >> n;
	for (int i = 0; i<n; i++) {
		cin >> array[i];
		for (int k = 0; k<i; k++) {
			if (array[i] == array[k]) {
				flag = 0;
			}
		}
		if (flag) {
			cout << array[i] << " ";
		}
		flag = 1;
	}
}
