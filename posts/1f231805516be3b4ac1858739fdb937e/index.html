laravel 路由404问题

今天在创建一个laravel 项目的时候，首页能够访问

Route::get(’/’, function () {
//return view(‘welcome’);
return “Hello word!”;
});

但是添加了一个路由访问发现一直访问不了

找了很多资料都没有解决这个问题，最后在vhosts.conf文件里面添加了一段就可以了

try_files $uri $uri/ /index.php?$query_string;

备注：我用的是phpstudy 的集成环境（php 7.1.13+nginx）
