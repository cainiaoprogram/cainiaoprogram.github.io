getrange命令

Redis getrange， 命令用于获取存储在指定 key 中字符串的子字符串。
字符串的截取范围由 start 和 end 两个偏移量决定(包括 start 和 end 在内)。

语法

getrange key start end

返回值

截取得到的子字符串

例子

127.0.0.1:6379> set test:getrange "long live China."
OK

127.0.0.1:6379> getrange test:getrange 0 -1
"long live China."

127.0.0.1:6379> getrange test:getrange -6 -1
"China."
