Java 常用开发工具配置之Eclipse

eclipse是常用IDE中的一款，历史悠久对于大多数Java程序员，人生的第一行代码想必就是由他编辑出来的，今天总结一下工作中经常使用到的配置。

  2. 将features目录下所有文件复制到，eclipse或myeclipse安装目录下对应的features文件夹下，例如我本地的目录 ：D:\Program Files\eclipse noen\features 注：如提示是否覆盖，选择覆盖

二、配置

1.编码相关配置

智能提示：工具栏 Window–>Preferences–>Java–>Editor–>Content Assist

输入：26位英文字母即abcdefghijklmnopqrstuvwxyz，点击Apply。

自动格式化代码：即进行保存操作后将代码自动格式化。

工具栏 Window–>Preferences–>Java–>Editor–>Save Actions

勾选 Perform the selected actions on save 和 Format source code
单选按钮 按需选择，点击Apply即可。

修改字体大小及样式：

工具栏 Window–>Preferences–>General–>Appearance–>Colors and Fonts

选择 Basic，拉动滚动条选择Text Font，点击Edit

选择自己喜欢的大小样式，点击确定

点击Apply即可
拼写检查：在编码过程中，我们缩写一些单词或者书写某种意义的字符，下方经常会出现红色或者黄色的波浪线，强迫症患者很是难受，只要关闭拼写检查这个问题就迎刃而解了。

工具栏 Window–>Preferences–>General–>Editors–>Spelling

去除勾选，点击Apply and Close

背景颜色：长时间使用电脑，希望大家也要爱护自己的眼睛，个人很喜欢护眼色作为背景颜色，具体实现如下：

工具栏 Window–>Preferences–>General–>Editors–>Text Editors

选中 Text Editors，右边出现Test Editors面板。 找到Appearance color options选项，选中Background color，去除System Default勾选，点击Color，在弹出的窗口选择规定自定义颜色，在色调选项卡中输入：82，123，205，点击添加到自定义颜色


点击确定，再点击Apply即可。

修改Console背景颜色：

工具栏 Window–>Preferences–>Run/Debug–>Console

找到background color 选项卡，点击颜色方框，在弹出的窗口中选择颜色，也可以自定义颜色，具体操作同上；

修改Package Explore 背景颜色：

找到eclipse安装目录下plugins文件夹 例如：D:\Program Files\eclipse sts\plugins

找到名称开头以org.eclipse.ui.themes这样给文件夹，例如：org.eclipse.ui.themes_1.2.1000.v20200528-1125

进入org.eclipse.ui.themes_1.2.1000.v20200528-1125\ 目录下：
修改e4_basestyle.css文件， 添加如下代码：（护眼色）

#org-eclipse-jdt-ui-PackageExplorer Tree,
#org-eclipse-ui-navigator-ProjectExplorer Tree,
#org-eclipse-ui-views-ContentOutline Tree,
#PerspectiveSwitcher ToolBar {
font-family: courier new;
font-size: 10px;
background-color: RGB(199,237,204);

保存并重启eclipse。
注： 默认eclipse主题样式，修改e4_basestyle.css文件，黑色对应dark，白色对应light。
