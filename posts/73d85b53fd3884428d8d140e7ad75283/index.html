VS Code mac “检测到#include错误。请更新includePath。”的解决方法

最近vs code里Java和C一起写，但有的时候会出现“检测到#include错误。请更新includePath。”的问题，在csdn上看了很多解决方法好像没卵用…自己的解决方法如下：

前提条件
  1. 报错内容为“检测到#include错误。请更新includePath。”
  2. 打开Finder -> Macintosh HD -> Library -> Developer -> CommandLineTools -> usr -> lib，看看里面有没有名叫clang文件夹。如果没有，以下解决方案可以使用；如果有，该解决方法不适用。
解决方法
  1. 打开terminal
  2. 输入
xcode-select --install
  1. 等待安装
  2. 安装完后回到Finder -> Macintosh HD -> Library -> Developer -> CommandLineTools -> usr -> lib看是否有clang文件夹。如果有的话，继续下一步。
  3. 进入clang文件夹，clang -> 版本号 -> include，右键点Get Info -> 复制Where的内容
  4. 返回vs code，用快捷键command+P查找名为“c_cpp_properties.json”的文件
  5. 将第五步复制的内容加上“/include”粘贴至"includePath"下
"includePath":[
	"${粘贴内容/include}/**"
]
  1. command+S
  2. 返回之前的代码，确认是否能运行。
