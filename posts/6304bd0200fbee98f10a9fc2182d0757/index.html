Java获取当前时间的小时/分钟

java获取当前时间的小时/分钟等。。。

方法一：

Date date = new Date();
int hours = date.getHours();


方法二：

GregorianCalendar calendar = new GregorianCalendar();
int hour = calendar.get(Calendar.HOUR_OF_DAY);
System.out.println("hour=" + hour);
