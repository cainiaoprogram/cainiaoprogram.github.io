rz命令安装和使用——Linux-CentOS

今天需要把Windows系统中的文件上传到VM中Linux里。

1.rz

首先命令行输入rz并回车发现提示未找到命令，那接下来就开始安装。

2.yum install lrzsz -y

输入yum install lrzsz -y回车，等待一小会，就提示已安装完毕。

3.rz

安装完成后输入rz回车，弹出选择文件对话框，双击打开。

 上传完毕后，点关闭。
