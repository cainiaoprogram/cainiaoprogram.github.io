Difference between getc(), getchar()， and gets()

getc(): 从输入中只能读单个字符
getchar()：从标准输入流中输入都单个字符。
两者基本等同，唯一不一样的是getc()是任何输入流，而getchar()是标准输入流。

gets:可以读入含有空格的字符串

// Example for getc() in C
#include <stdio.h>
int main()
{
    printf("%c", getc(stdin));
    return (0);
}


// Example for getchar() in C
#include <stdio.h>
int main()
{
    printf("%c", getchar());
    return 0;
}
