微软x86服务器,x86服务器带动微软Server攫取市场份额

作者：中关村在线 王迪

根据国外报道，近日IDC最新报告显示，大多数企业正在使用运行Windows Server的x86服务器来处理核心业务。针对这样的一种市场转变，IDC报告中提出转变的原因很多，但主要是得益于x86服务器硬件技术的完善以及Windows Server 2008 R2和SQL Server 2008 R2的大量改进。

另外，IDC在进行这份调查后结果显示，企业的业务工作量正逐渐由基于RISC和EPIC(基于Itanium)架构的大型主机转向x86服务器。这些变化得益于微软在产品上的战略转变，即在下一代Windows Server、SQL Server 2008 R2和Visual Studio 2010中不在支持Intel安腾架构。

值得关注的是，IDC从新规范了中端服务器市场，价格在2.5万至25万美元之间的服务器产品列入到中端产品行列，由此基于Windows Server 2008 R2已经成为x86服务器更新硬件的必选系统。同时对比2008年部署操作系统数据，IBM的z/OS占据首位，份额约为70%，其次为份额为44%的Unix、份额为22%的Windows和18%的Linux。

然而，数据中显示出，在2009年中高端Unix服务器出现了销量和营收的下滑，如今x86服务器占据了90%以上市场销量，全球超过50%的服务器收益都来自于x86服务器，随着这种情况的转变，微软会利用x86服务器市场时机，Windows Server 2008 R2将在数据中心领域获得更多的市场份额。
