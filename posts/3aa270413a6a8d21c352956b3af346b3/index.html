C/C++：gcc选项-fno-access-control的使用

在测试代码时需要访问私有和保护成员，使用这个选项可以忽略访问控制。

class A
{
public:
    A() {}
protected:
    void test_protected() {}
private:
    void test_private() {}
};

int main(int argc, char* argv[])
{
    A a;
    a.test_protected();
    a.test_private();
    return 0;
}

$ g++ main.cpp -fno-access-control
