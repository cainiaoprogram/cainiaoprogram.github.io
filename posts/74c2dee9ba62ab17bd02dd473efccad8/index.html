建立共享文件夹

  1. PC机新建共享文件夹
    在 Windows 系统的中创建 [任意名]文件夹，记住路径。
  2. 移除虚拟机原有共享文件夹
    在 VMware 虚拟机界面点击“虚拟机”，弹出选项后选择“设置”进入虚拟机设置界面，点击“选项”–>“共享文件夹”–>“总是启用”–>“文件夹（F）”中移除无法使用的共享路径。

交叉编译

查看是否安装了交叉编译工具

which arm-linux-gccgcc

安装交叉编译工具链

  1. 在 Windows 系统中，将压缩包 复制到 ubuntu 共享目录夹中，然后linux进入该目录，执行解压命令。
    进入目录：
cd /mnt/hgfs/share

解压命令：

sudo tar Jxvf arm-linux-gnueabi-5.4.0.tar.xz -C /

C 后面有个空格，并且 C 是大写的，它是英文单词“Change”的第一个字母，在此是改变目录的意思。
2. 把编译器路径加入系统环境变量，运行命令
先进入root模式

sudo -i   （这是我进入root模式的命令）
cd /etc/ 
vim /etc/bash.bashrc

**在.bashrc 文件最后一行加上 （该路径要与实际安装路径一致,即/usr/local/arm/5.4.0/usr/bin 这个路径要在你的 Ubuntu 中找的到，要是实际存在的路径）
**

export PATH=/usr/local/arm/5.4.0/usr/bin:$PATH

保存退出后，执行以下操作：

source /etc/bash.bashrc （立即使新的环境变量生效，不用重启电脑）

检查是否将路径加入到 PATH，执行以下命令

echo $PATH

若显示的内容中有刚刚添加的交叉编译器所在目录，则证明编译环境安装成功。
3. 测试是否安装成功

arm-linux-gcc -v

出现以下信息，表示交叉编译工具链安装成功：

多窗口终端–Terminator

Ubuntu下载软件都是从软件源获取的。
在终端输入更新软件源的指令：

sudo apt-get install

安装指令：

sudo apt-get install terminator


如果没有显示–采用搜索添加（右键添加收藏夹）
