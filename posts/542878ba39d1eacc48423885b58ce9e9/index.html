安装openssh-server提示依赖包问题

我们在进行远程文件传输的时候，经常会使用ssh传输，那么我们需要安装openssh-server，可是在安装的时候会遇到依赖包问题，如下： 
下列软件包有未满足的依赖关系：
 openssh-server : 依赖: openssh-client (= 1:6.6p1-2ubuntu)
E: 无法修正错误，因为您要求某些软件包保持现状，就是它们破坏了软件包间的依赖关系。

这是因为,openssh-server是依赖于openssh-clien的，原由是自带的openssh-clien与所要安装的openssh-server所依赖的版本不同,这里所依赖的版本是

= 1:6.6p1-2ubuntu1

所以要安装对应版本的openssh-clien,来覆盖掉ubuntu自带的

apt-get install openssh-client=1:6.6p1-2ubuntu1

然后安装openssh-server

apt-get install openssh-server

最后显示

Setting up ssh-import-id (5.5-0ubuntu1) ...
Processing triggers for systemd (229-4ubuntu7) ...
Processing triggers for ureadahead (0.100.0-19) ...
Processing triggers for ufw (0.35-0ubuntu2) ...

openssh-server安装成功
