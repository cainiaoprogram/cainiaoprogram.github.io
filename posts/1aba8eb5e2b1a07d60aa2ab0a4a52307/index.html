python 提取字符串中的数字组成新的字符串

　　

　　方法一

 

# 有一个字符串text = "aAsmr3idd4bgs7Dlsf9eAF"
# 请将text字符串中的数字取出，并输出成一个新的字符串
import re

text = "aAsmr3idd4bgs7Dlsf9eAF"
text = re.sub("\D", "", 'aAsmr3idd4bgs7Dlsf9eAF')
print(text)

 

 

　　方法二

# 有一个字符串text = "aAsmr3idd4bgs7Dlsf9eAF"
# 请将text字符串中的数字取出，并输出成一个新的字符串

import re

text = "aAsmr3idd4bgs7Dlsf9eAF"
text = ''.join(re.findall("\d+",text))
print(text)
