查看网卡名、ip地址、网关

首先查看网卡名、ip地址、网关，找到对应的网卡名并记录其地址

ifconfig


route -n

切换root用户，进入配置文件

sudo -i

cd /etc/netplan

 vim 01-network-manager-all.yaml

配置静态IP


每个人的系统参数不同，仔细对照自己的来修改

# Let NetworkManager manage all devices on this system
network:
        ethernets:
            ens33:
                    addresses: [192.168.85.142/24]   #ip
                    gateway4: 192.168.85.2
                    dhcp4: no
                    nameservers:
                             addresses: [114.114.114.114]

        version: 2
        renderer: NetworkManager

配置完退出后，执行netplan apply，没问题即配置完成
运行报错，大多数情况都是层次不对、格式不对、没有空格、名称没写对，请仔细检查
