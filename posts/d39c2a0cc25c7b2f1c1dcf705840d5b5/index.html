前言

车载以太网是一种用以太网连接车内电子单元的新型局域网技术。与普通的以太网使用4对非屏蔽双绞线（UTP）电缆不同，车载以太网在单对非屏蔽双绞线上可实现100Mbit/s甚至1Gbit/s的数据传输速率，同时还应满足汽车行业对高可靠性、低电磁辐射、低功耗、带宽分配、低延迟以及同步实时性等方面的要求。
车载以太网的物理层采用了博通公司的BroadR-Reach技术，BroadR-Reach的物理层（PHY）技术已经由单线对以太网联盟（One-pair Ethernet Alliance，OPEN）标准化，因此有时也称车载以太网为BroadR-Reach（BRR）或OABR（Open Alliance BroadR-Reach）。
本章节就主要介绍车载以太网物理层如何测试。

一、主/从模式测试

测试目的：验证车载以太网 PHY 的 Master/Slave 配置是否符合需求规范。
前置条件：DUT正常工作
测试步骤：
1、设置DUT供电电压 𝑉𝑏𝑎𝑡 为 𝑉𝑁𝑂𝑅𝑀𝐴𝐿；
2、DUT 上电，DUT 处于持续唤醒状态；
3、设置 Link Partner 为 Slave 模式；
4、读取 DUT 的 Link 状态，DUT Link 状态应符合需求规范为 Link Up/Link Down；
5、设置 Link Partner 为 Master 模式；
6、读取 DUT 的 Link 状态，DUT Link 状态应符合需求规范为Link Down/Link Up。

二、主/从模式配置

测试目的：验证车载 以太网PHY 是否可以配置 Master/Slave 模式。
前置条件：DUT正常工作
测试步骤：
1、设置DUT供电电压 𝑉𝑏𝑎𝑡 为 𝑉𝑁𝑂𝑅𝑀𝐴𝐿；
2、DUT 上电，DUT 处于持续唤醒状态；
3、设置 DUT 处于 Master 模式；
4、CANoe读取 DUT 的 PHY状态，DUT 为 Master 状态；
5、设置 DUT 处于 Slave模式；
6、CANoe读取 DUT 的 PHY状态，DUT 为 Slave 状态。

三、Link-up/ Link-down时间

测试目的：验证 PHY Link-up/ Link-down时间是否符合需求规范。
前置条件：DUT为休眠状态
测试步骤：
1、CANoe向CAN总线发送CAN NM报文，记录时间为T1；
2、等待DUT Link Up，记录时间为T2；
3、T2-T1为ETH Link up时间；
4、CANoe停止发送CAN NM报文，记录时间为T3；
5、等待DUT Link down，记录时间为T4；
6、T4-T3为ETH Linkdown时间。

四、低压通信范围测试

测试目的：测试 DUT 低电压通讯范围是否符合需求规范。
前置条件：DUT正常工作
测试步骤：
1、设置 DUT 供电电压为 12V；
2、本地唤醒 DUT 后等待 30s 至总线通信稳定；
3、等待DUT发出任意以太网报文；
3、从12V开始以0.5V为步进降低电压，并发出icmp ping报文在当前电压下持续10s；
4、当电压下降至9V时，以0.1为步进继续下降电压；
5、当DUT未响应icmp ping报文，记录当前电压为停止通信电压，DUT 在电源电压降低到6.5V 及以下时允许停止以太网通讯；
6、以通信停止电压为起始，以0.1步进增加电压，并发出icmp ping报文在当前电压下持续10s；
7、当DUT响应icmp ping报文，记录当前电压为恢复通信电压，在电压恢复到 6.5V 及以上时必需能够恢复通讯。

五、高压通信范围测试

测试目的：测试 DUT 高电压通讯范围是否符合需求规范。
前置条件：DUT正常工作
测试步骤：
1、设置 DUT 供电电压为 12V；
2、本地唤醒 DUT 后等待 30s 至总线通信稳定；
3、等待DUT发出任意以太网报文；
4、从12V开始以0.5V为步进增加电压，并发出icmp ping报文在当前电压下持续10s；
5、当电压上升至16V时，以0.1为步进继续增加电压；
6、当DUT未响应icmp ping报文，记录当前电压为停止通信电压，DUT 在电源电压升高到18V 及以上时允许停止以太网通讯；
7、以通信停止电压为起始，开始测试通信恢复电压。每次以0.1步进减少电压，并持续10s发出icmp ping报文；
8、DUT响应icmp ping报文，记录当前电压为恢复通信，在电压恢复到18V 及以下时必需能够恢复通讯。
