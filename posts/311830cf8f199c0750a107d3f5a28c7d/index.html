ElementUi的侧边栏操作

相信新手玩家在接触ElementUi的时候总是很难读懂ElementUi的一些基本知识

本文章主要讲述的是侧边栏如何能够伸展覆盖整个左侧

.Menu {
  .el-aside {
    height: 100%;
    .el-menu {
      height: 100%;
    }
    .el-submenu .el-menu-item {
      min-width: 0;
    }
  }
}

加上这些样式就可以使得左边侧边栏完全展开（Menu是最外层div盒子的命名）
