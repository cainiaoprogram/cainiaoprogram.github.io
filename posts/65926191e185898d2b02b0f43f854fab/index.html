MFC 控件添加热键

   
   
  1. 转载自https://www.cnblogs.com/hungryvampire/p/5297927.html
  2. MFC 控件添加热键 2014-12-24 14:28:47
  3. 标签：C++ MFC 控件 热键使用 
  4.     给MFC中的控件添加我们想要的控件热键，在动手之前，必须清楚，热键分为local的和global的， 其中local的职能在当前程序有焦点（被激活）时有效，而global的，则无论什么时候都有效，测试local的要优先于global的，就是如果当前激活窗口的快捷键与未激活窗口的快捷键重叠，当前激活窗口优先响应。另外还包括menu，button。
  5.     自然而然，创建热键的方法也有多种，不同的创建方法创建的热键作用范围不一定相同。应该根据需求合理的选择自己的方法。
  7. 方法一：
  9.     打开对话框资源，选择指定控件的属性-->在caption项中你定义的名字后添加（&Y）。这样就可以实现热键。其中Y表示你要制定的快捷按键，我选的是Y，按下ALT+Y即可执行这个控件。
  11.     该方式热键只有当FOCUS在控件上时才起作用，算是局部热键。
  13. 方法二：
  15. 1. 在资源视图中添加资源，选择"Accelerator"选项新建即可；
  16. 2. 打开新建的文件夹下的"IDR_ACCELERATOR1"中编辑，其中包括ID、修饰符、键、类型。ID中选择你要添加快捷键的菜单、按钮的ID编号，修饰符中选择你是否要使用组合键，如：Alt、Alt+Shift等等，键中选择你要使用什么样的虚拟键码，类型中选择VK形式或者是ASCII形式。
  17. 3. 在你的对话框的头文件中添加快捷键变量，如HACCEL m_hAcc；
  18. 4. 在你的对话框的源文件的OnInitDialog函数中加载快捷键，如：
  19. 1 HACCEL m_hAcc=LoadAccelerators(AfxGetInstanceHandle(),MAKEINTRESOURCE(IDR_ACCELERATOR1)); 
  21. 5. 重载函数PreTranslateMessage，使用类向导，重载虚函数PreTranslateMessage，如：
  22. 12345678910 BOOL CVideoMonitorDlg::PreTranslateMessage(MSG* pMsg) 
  23. { 
  24.     if (WM_KEYFIRST<=pMsg->message&&pMsg->message<= WM_KEYLAST)  
  25.     {  
  26.         HACCEL hAccel=m_hAcc;  
  27.         if (hAccel && ::TranslateAccelerator(m_hWnd, hAccel, pMsg))  
  28.         return TRUE; 
  29.     }  
  30.     return CDialogEx::PreTranslateMessage(pMsg); 
  31. } 
  33. 6. 通过以上5个步骤基本上即可对工程中的菜单或者按钮添加热键。
  34. 方法三：
  35. 使用WM_HOTKEY。
  36. 在对话框头文件中：
  38. 12 afx_msg LRESULT OnHotKey(WPARAM wParam,LPARAM lParam); 
  39. afx_msg void OnDestroy(); 
  41. 在对话框CPP文件中：
  43. 1234 BEGIN_MESSAGE_MAP(CYourDlg, CDialog) 
  44.    ON_MESSAGE(WM_HOTKEY,OnHotKey)  
  45.    ON_WM_DESTROY() 
  46. END_MESSAGE_MAP() 
  48. hotkey对应映射：
  50. 1234567 LRESULT CYourDlg::OnHotKey(WPARAM wParam,LPARAM lParam) 
  51. { 
  52.     if（wParam==IDC_XXX） 
  53.         OnYourFunction(); 
  54. //ToDo: add function 
  55.      return 0;  
  56. } 
  58. 注册热键：
  60. 1234 BOOL CYourDlg::OnInitDialog() 
  61. { 
  62.     ::RegisterHotKey(GetSafeHwnd(), IDC_XXX, MOD_ALT, 'D');//注册热键alt+D（D必须大写） 
  63. } 
  65. 记得销毁：
  67. 1234 void CYourDlg::OnDestroy() 
  68. { 
  69.     ::UnregisterHotKey(GetSafeHwnd(),IDC_XXX);//销毁热键 
  70. }
