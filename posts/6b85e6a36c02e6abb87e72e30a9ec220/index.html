Gscan2pdf 2.5.5发布了，Gscan2pdf是用于从扫描文档生成PDF/DjVu的GTK工具，近日发布了bug修复，一些改进和更新的翻译。

Gscan2PDF

2.5.5更改日志：

清除所有页面时修复偶尔的错误消息

如果不应用配置文件，仅将纸张设置为手动

更新带有和不带OCR输出的页面之间的OCR视图切换

如果xz可用，则xz压缩日志文件

修复配置文件设置SANE_INFO_INEXACT后应用纸张的错误

将当前设置保存为新配置文件时，实际设置配置文件

如果设备在打开时抛出错误，请将其添加到会话黑名单中

选项容差，以覆盖返回不精确选项而不设置SANE_INFO_INEXACT的错误后端

更新翻译。

如何在Ubuntu中安装gscan2pdf 2.5.5：

官方Ubuntu PPA

为Ubuntu 16.04，Ubuntu 18.04，Ubuntu 18.10，Ubuntu 19.04及其衍生产品制作了新的发布包。

1.通过在键盘上按Ctrl + Alt + T或从软件启动器搜索“终端”来打开终端。

打开时，运行

命令

添加PPA：

linuxidc@linuxidc:~/www.linuxidc.com$ sudo add-apt-repository ppa:jeffreyratcliffe/ppa

在提示并按Enter键时键入用户密码(由于安全原因没有星号反馈)。

2.对于已安装先前版本的用户，请通过软件更新程序进行升级。

您可以通过Synaptic Package Manager或在终端中运行以下

命令

来首次安装它：

linuxidc@linuxidc:~/www.linuxidc.com$ sudo apt-get update

linuxidc@linuxidc:~/www.linuxidc.com$ sudo apt-get install gscan2pdf

如何删除：

要删除gscan2pdf，请使用系统包管理器或运行命令：

linuxidc@linuxidc:~/www.linuxidc.com$ sudo apt-get remove --autoremove gscan2pdf

可以通过“软件和更新”实用程序在“其他软件”选项卡下删除PPA。
