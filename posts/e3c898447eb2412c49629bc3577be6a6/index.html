nginx--中文域名解析

项目场景：

各大域名供应商目前已出现中文域名，记录中文域名在Linux环境下的配置。

解决方案：

  • 在站长之家的进行中文域名转码

  •  在nginx.conf中进行域名配置
  • server {
    listen 80;
    server_name xn--0zwm56d.xn--fiqs8s;
    ...
}
