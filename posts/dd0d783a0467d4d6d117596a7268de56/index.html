dedecms横向登录

1、在index.htm中加入js代码

 
  
  1. <!--横向登录开始--> 
  2. <script language="javascript" type="text/javascript" src="{dede:global.cfg_cmsurl/}/include/dedeajax2.js"></script>   
  3. <script language="javascript" type="text/javascript">   
  4. function CheckLogin(){    
  5. var taget_obj = document.getElementById('_userlogin');    
  6.       myajax = new DedeAjax(taget_obj,false,false,'','','');    
  7.       myajax.SendGet2("{dede:global.cfg_cmspath/}/member/ajax_loginsta.php");    
  8.       DedeXHTTP = null;    
  9.     }    
  10. </script> 
  11. <!--横向登录结束--> 
 
  
2、在head.htm中加入js代码:
 
  
 
  
  1. <script language="javascript" type="text/javascript">CheckLogin();</script> 
 
  
来判断是否登录
 
  
3、修改member/ajax_loginsta.php文件，该文件是显示登录成功后的样式。
