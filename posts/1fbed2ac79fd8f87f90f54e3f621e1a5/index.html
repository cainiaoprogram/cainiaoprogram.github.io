数字翻转（4位数）

任务描述

本关任务：输入一个四位正整数，输出这个数的翻转。

测试说明

平台会对你编写的代码进行测试：

测试输入：4233；

预期输出：

3324

测试输入：5211；

预期输出：

1125

#include<stdio.h>

int main()
{
	int a;
	scanf("%d",&a);
	
	int b=a%10;
	int z=a/10;
	int c=z%10;
	int y=z/10;
	int d=y%10;
	int x=y/10;
	
	int q=b*1000+c*100+d*10+x;
	printf("%d",q);
	
	return 0;
	
}
