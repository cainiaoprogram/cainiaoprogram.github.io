4 阿里云鉴权账号配置

url：快速入门（开通和使用）

新建一个文件夹，用于存放fl_client.ini配置文件，我放在存在安装文件的同一目录。

[security]

# key for client authentication

aliyun_license_key = xxx #请修改成您购买服务的LicenseKey信息，见控制台的已购服务列表（开通服务中描述）

aliyun_ak_id= xxx #请修改成您阿里云账号的AccessKey信息

aliyun_ak_secret= xxx #请修改成您阿里云账号的AccessKey Secret信息

#以下内容复制后，无特殊情况不需要更改

[server]

# license server host

name = opt.cn-beijing.aliyuncs.com # 请修改为您购买区域的endpoint，当前仅开放的北京区，不需要修改。

# license server port 80 if absent

port = 443

# license server is https protocol

https = true

# client side request timeout in second

request_timeout = 2

修改地方3处

1 aliyun_license_key = xxx #请修改成您购买服务的LicenseKey信息，见控制台的已购服务列表（开通服务中描述）

就是这幅图里面的LicenseKey。

2 aliyun_ak_id= xxx #请修改成您阿里云账号的AccessKey信息

3 aliyun_ak_secret= xxx #请修改成您阿里云账号的AccessKey Secret信息

创建访问密钥请在AccessKey管理创建。

建议设置环境变量MINDOPT_LICENSE_PATH，将fl_client.ini文件路径添加进去。对应官方文档链接鉴权文件放置位置：
