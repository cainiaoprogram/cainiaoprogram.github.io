Mac android studio && Xcode 快捷键

快捷键总结，随时更新

Mac android studio 常用快捷键

1、command + delete         删除一行

2、command + option + L  格式化代码

3   control + option + i        格式化代码

4、shift + shift                  查找文件或者类

5、command + shift + O       查找文件

6、option 然后拖住鼠标左键   多行操作

7、shift + F6                   重命名


Xcode 常用快捷键

1、control + I (i)               格式化代码

2、option 然后拖住鼠标左键  多行操作

3、command + shift + O        查找文件

4、command + option + [ / ]    上下移动

5、 command + ]  / [       左右移动

6、command + control + e     全局修改局部变量快捷键

7、 command + shift + f             快速打开全局查询

8、command + control + up/down  快速切换到.m文件/.h文件

9、command + up/down             快速切换到文件的顶部/底部

10、command + delete               删除光标所在行之前的代码

11、option + delete                   删除光标之前的一个单词

12、command + 左 / 右               将光标移动到所在行的最前面/后面

13、option + 左 / 右                  将光标移动到所在单词的最前面/后面


Mac 快捷键

1、command + shift + .                         隐藏的文件显示出来
