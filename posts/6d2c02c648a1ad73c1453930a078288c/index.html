从服务器自身提供的 DNS 域名解析故障

connection timed out; trying next origin ;; connection timed out; no servers could be reached
修改bind服务器named.conf配置即可，将
options {
listen-on port 53 { 127.0.0.1; };
listen-on-v6 port 53 { ::1; };
修改为
options {
listen-on port 53 { 192.168.1.8; }; ####本机ip
listen-on-v6 port 53 { ::1; };
再将dns地址参数修改成 192.168.1.8
[root@node01 slaves]# cat /etc/resolv.conf
nameserver 192.168.1.8
测试：
[root@node01 slaves]# nslookup
