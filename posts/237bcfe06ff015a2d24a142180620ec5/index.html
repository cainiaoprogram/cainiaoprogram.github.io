Nginx的启动与关闭命令

目录：

    • 一、前言
    • 二、命令

一、前言

nginx如果全局安装则可以如下命令执行，否则需要进入nginx的sbin下执行，都带上./

二、命令

  • 启动: 直接使用命令: nginx
nginx
  • 关闭1: 快速停止
nginx -s stop
  • 关闭2: 完整有序停止
nginx -s quit
  • 重启: 如下
nginx -s reload
