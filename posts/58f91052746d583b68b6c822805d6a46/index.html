node、Vue cil、vs code安装

一、node安装

win7安装Node需要注意版本问题，最新版本不支持win7，需要使用较低的版本。如：12版

node的官方网站：https://nodejs.org/en/download/

下载完成后直接下一步直到安装结束。

二、vue cil安装

安装Vue cil前需要安装好node，使用命令node --version在cmd中查看

 运行命令：npm install -g vue-cil

三、安装VS code

在官网下载相应的安装包，一直下一步直到安装结束。可自行选择路径
