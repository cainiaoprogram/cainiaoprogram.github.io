Gitlab-CI 备忘录

1.编译环境
    将安装包拷贝到/opt目录下 ,在~/.bashrc配置环境变量
        #set java environment
        export JAVA_HOME=/opt/jdk1.8.0_202 
        export JRE_HOME=${JAVA_HOME}/jre
        export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
        export PATH=${JAVA_HOME}/bin:$PATH

        #set android environment
        export ANDROID_HOME=/opt/android-sdk-linux
        export PATH=$ANDROID_HOME/tools:$ANDROID_HOME/platform-tools:$PATH

        #set android environment
        export GRADLE_HOME=/opt/gradle-4.10.1
        export PATH=$GRADLE_HOME/bin:$PATH

        #set NDK environment
        export NDK_HOME=/opt/android-ndk-r18b
        export PATH=$NDK_HOME:$PATH

        #set 360jiagu environment
        export JIAGU_DIR=/opt/360jiagubao_linux_64/jiagu

    安装SDK 所需要32位包 sudo apt-get install -y libc6-i386 lib32stdc++6 lib32gcc1 lib32ncurses5 lib32z1
    测试环境变量： java -v，ndk-build -v , gradle, adb shell .
    安装Git并配置GIt SSH 公钥：
        cd ~/.ssh  ssh-keygen -t rsa生成ssh的公钥和私钥 ,将公钥拷贝到gitlab配置
2.安装Runner
    第一次切换root 用户先执行sudo passwd root
    sudo apt-get update
    sudo apt-get install gitlab-ci-multi-runner
    如果找不到，切换163数据源
3.注册Runner
    gitlab-ci-multi-runner register
    注册时的url 是gatlab对应项目的url token是gatlab对应的token 在gatlab项目下的setting runner里面可以看到
4.Run
    gitlab-ci-multi-runner run-single --url https://git.bg.wanduoduo.com/ --token 1e82b0c3eafccf02faefe1c82c4d52 --executor shell
    这里面的token 不是注册时候的token 是注册后生成的新token
5.安装Fir-cli
    sudo apt install ruby  先安装ruby
    sudo apt-get install   安装ruby-dev
    sudo gem install fir-cli 安装fir-cli

6.制作Docker镜像
   创建Dockerfile文件并编写内容（需单独创建一个文件夹放dockerfile文件，和需要的资源）
   docker build -t dockerfile .


PS:
    编译Android项目需要下载依赖，如果不能翻墙下载，需要配置国内的阿里镜像：
        在项目根目录的build.gradle文件里面加上如下代码 maven{ url 'http://maven.aliyun.com/nexus/content/groups/public/'}
