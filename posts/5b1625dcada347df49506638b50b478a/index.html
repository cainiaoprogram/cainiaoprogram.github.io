创建docker镜像并push到docker hub上

首先在上一篇文章中我已经安装好了redis6.0及gcc7.1。
先同步一下系统和硬件的时间，要不然docker hub会出错
同步命令：hwclock
创建容器：docker build -t redis50 .
查看容器：docker images
运行容器：docker run -itd --name redis50 redis
查看容器信息：docker ps
进入容器：docker exec -it redis50 bash
退出容器：exit
