【Qualcomm】在EFS的NV项中打开voice通话自动接听

1. set NV 5280 to 0

2. set NV 74 to 1

3. 重做EFS烧写验证即可。

【Qualcomm】在EFS的NV项中关闭VoLTE高清语音服务

VoLTE功能开关：
NV67218 = 1 enable（IMS）

NV67218 = 0 disable（IMS）

重新制作EFS即可。
