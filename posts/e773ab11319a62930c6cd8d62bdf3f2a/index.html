解决IDEA上传Git 带有本地编译文件问题

IDEA 版本控制推送代码问题

    IDEA开发过程中，使用版本控制Git、Svn或其他版本管理工具提交代码到远程经常遇到一些不重要的文件。这些文件是本地的文件，不需要上传，这个时候就需要通过设置，来改变。

问题解决

在IDEA通过设置，忽略这些文件提交代码时出现的文件
步骤

进入idea设置界面
    路径：
    File 》Settings 》Editor 》 File Types
    在“ignore files and folders” 框里加入 *.iml;.idea;target;
