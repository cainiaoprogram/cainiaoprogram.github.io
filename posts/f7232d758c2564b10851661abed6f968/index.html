三、创建虚拟环境，安装tensorflow2.0

创建虚拟环境：

conda create -n tensorflow python=3.6

激活环境：

activate tensorflow2.0

安装tensorflow2.0：

pip install tensorflow-gpu==2.0.0a.0 numpy==1.16.4 -i https://pypi.tuna.tsinghua.edu.cn/simple
