vb.net中的整除
C#中的除法的写法应该很简单，不过 vb.net中的除法就不是简单的一个 / 了，在vb.net中存在两种除法的写法，一种是 / 普通除法的写法它允许小数的存在而且会自动的四舍五入，而 / 这中写法就是vb.net中的整除了。举例如下
 
c#: int a=2;
      int b=3;
      int c=0;
      c = b / a;
输出结果 1
 
vb.net:
         dim a as integer = 2
         dim b as integer = 3
         dim c as integer = 0
         c=b/a
         c=b/a
输出结果:
          2
          1
