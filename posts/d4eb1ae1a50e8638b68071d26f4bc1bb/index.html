C/C++ 知识点：初始化操作由 case 标签跳过

一、初始化操作由 case 标签跳过

如果要在case里面定义变量，需要用括号括起来{}，不然会出错，例如：

#include <iostream>
using namespace std;

void main()
{
    int val = 2;
    switch(val)
    {
    case 1:
        int x = 20;
        break;
    case 2:
        break;
    default:
    }
}

上面的代码在编译时就会出现下面的错误

main.cpp(15): error C2360: “x”的初始化操作由“case”标签跳过
