el-table 使用过滤器


如图，后端传来的数据是0，1 我们需要把数组展示为 ‘是’,‘否’

关键代码（html）：

<el-table-column prop="sf" label="是否参数代理" align="center">
    <template slot-scope="scope">{{ scope.row.sf | csdlFilter }}</template>
</el-table-column>

（js）:

filters: {
     csdlFilter(val){
         if(val == '0'){
             return '否'
         }else if(val == '1'){
             return '是'
         }else{
             return '-'
         }
     }
 }
