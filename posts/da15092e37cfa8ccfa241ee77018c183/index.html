Charles iOS抓https包

1、iPhone 连mac IP 代理

2、Charles -> Help -> SSL Proxying -> Install Charles Root Certificate -> 钥匙串里找到Charles证书点击完全信任

3、Charles -> Help -> SSL Proxying -> Install Charles Root Certificate on a Mobile Device orRemote Browser -> 弹框 -> OK

4、iPhone -> Safari -> chls.pro/ssl 下载证书，安装证书

5、Charles -> Proxy-SSL -> Proxying settings -> Add -> port ->443

6、ps iOS新版本系统无法抓到https包，手机通用-关于本机-最下面“证书信任设置”，找到Charles proxy 把开关打开就可以了
