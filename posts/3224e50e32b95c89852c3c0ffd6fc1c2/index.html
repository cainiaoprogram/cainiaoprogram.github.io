cadence 的常用快捷键

cadence 是一款电子设计自动化 (EDA) 软件，用于设计、模拟和分析电子电路。它提供了大量的快捷键，可以帮助用户更快速、高效地使用软件。

以下是一些常用的快捷键：

  • Ctrl + S：保存当前工程
  • Ctrl + Z：撤销
  • Ctrl + Y：重做
  • Ctrl + C：复制
  • Ctrl + V：粘贴
  • Ctrl + A：全选
  • Ctrl + F：查找
  • Ctrl + G：跳转到指定行
  • Ctrl + O：打开文件
  • Ctrl + P：打印
  • Ctrl + N：新建文件
  • Ctrl + W：关闭当前窗口
  • Ctrl + Q：退出 cadence
  • Ctrl + E：显示/隐藏属性编辑器
  • Ctrl + R：显示/隐藏对象树
  • Ctrl + T：显示/隐藏工具栏

这只是常用快捷键的一小部分，cadence 还有许多其他的快捷键，可以根据自己的需要来使用。
