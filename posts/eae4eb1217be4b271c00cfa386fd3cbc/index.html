ubuntu 源码编译安装make过程很慢问题解决

  1. 一般下载的时候有两种方式
  • 使用wget 下载后使用tar命令解压
  • 直接git clone 源代码
  1. 下载完成之后，进入目录，新建一个文件夹build，防止影响码源结构
  2. 第三步，进入到build目录，使用cmake编译 cmake ..
  3. 第四步直接make编译
  • 这一步有时候会很慢，我们使用make -jn
  • 这里n表示线程数，我一般使用八个线程 make -j8
  • 或者这里编译需要的软件即可 make clang -j8
  1. 最后安装 sudo make install
