 三、语法的嵌套（这个比较常用）

        使用Sass对于代码的结构非常清晰，具体如何使用下图可以进行表示

1、这是html页面的嵌套关系

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8" />
		<title></title>
	</head>
	<body>
		<div id="app">
			<span></span>
			<div class="app2">
				<a href="#" class="a"></a>
			</div>
		</div>
	</body>
</html>

 2、这是Sass页面如何进行的嵌套

#app{
	span{
		background-color: black;
	}
	.app2{
		.a{
			height: 20px;
		}
	}
}

 3、这是转换后css页面的代码

#app span {
  background-color: black; }

#app .app2 .a {
  height: 20px; }
