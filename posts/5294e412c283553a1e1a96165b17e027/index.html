斐讯N1刷无线打印服务器,斐讯N1刷armbian制作cups打印服务器支持airprint不使用foo2zjs|hp1020...

我自从买了N1以后就想着物尽所用，但是在网上查找了特别多的资料没有找到合适的，由于我的打印机是HP1020，网上所有支持HP1020的N1教程都不能用。如果你也尝试了很多，并且有缘分看到了我的这篇文章，你会发现真的很简单，但是走了很多弯路。网上很多教程都是类似的，很多都是抄来的，很多人都让你去用foo2zjs，其实并不一定好用，所以我不选择使用foo2zjs。

本教程适合几乎所有的armbian系统，同时支持debian和Ubuntu。

通过本教程，可以把老式的惠普打印机变成无线打印机，支持惠普几乎全系列的打印机。

第一步：

将你的N1刷成armbian系统，如果你看了其他教程还不会，可以等我出一个教程，其实我最不喜欢重复的造轮子所以就先不详细介绍了。

第二步：

通过ssh链接你的N1，如果你有无线键鼠而且还有独立的屏幕，那么用无线键鼠方案也可以，反正就是进入系统。

第三步：

更新一下

sudo apt-get update

sudo apt-get upgrade

第四步：

安装惠普支持套件，不选择使用foo2zjs。同时安装cups

sudo apt-get install hplip

sudo apt-get install hpijs-ppds

sudo apt-get install cups

第五步：

配置一下cups

sudo cupsctl --remote-admin --remote-any --share-printers

第六步：

重启一下N1，然后访问cups地址就好了

x.x.x.x:631

第七步：

登录(登录账号和密码与ssh的账号密码相同，就是你的管理员账号密码)网页以后，点击 Administrator》Add Printer》在Local Printers选择自己的打印机，然后一路向下(一定要选择share)然后在选择hpijs，一定要带hpijs的，不要选择cups的。

HP LaserJet 1020 hpijs, 3.18.12, requires proprietary plugin (en)

第八步：

ssh输入

hp-plugin

去cups的我的打印机里，打印一下测试页，如果看不懂可以使用网页翻译，其实都很简单的单词。

然后重启就行了。

后面的步骤就是在电脑上和手机上添加打印机了，ios打印一下就可以看到自己的打印机，电脑通过添加网络打印机，就可以顺利使用了。
