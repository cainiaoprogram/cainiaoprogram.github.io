vue项目兼容IE11

1、npm安装babel-polyfill

npm install babel-polyfill --save-dev

2、在入口文件main.js中引入

import 'babel-polyfill'

3、如果也是用了官方脚手架vue-cli，还需要在webpack.config.js配置文件中做修改，即可。

entry: {
  app: ["babel-polyfill", "./src/main.js"]
},

改后别忘记重启项目，清除下浏览器缓存，此方法兼容IE11亲测有效。
