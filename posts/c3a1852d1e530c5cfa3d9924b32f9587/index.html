ado 获取记录数

先看下面的 SQL 语句

 

SELECT COUNT(*) FROM task

 

如果用 ADO 获取返回的记录数？

 

其实很简单：

 

_variant_t var;

 

pRs->Open("SELECT COUNT(*) FROM task",
  m_pConnection.GetInterfacePtr(),
  adOpenStatic,
  adLockOptimistic,
  adCmdText);

var = pRs->GetCollect( 0L );
