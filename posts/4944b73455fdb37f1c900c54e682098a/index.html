Mac OS安装IDEA和JDK（超级详细教程）

一、首先安装IDEA（使用教育账号）

在官网进行下载：
Idea官网地址直接下载

我下载的是apple silicon的，这个取决于电脑，如果Intel的就下载上面的镜像

下载好了以后，直接点击下载好的那个包，就完成了。

二、下载安装JDK

到Oracle官网直接下载安装包：
官网网址：
SE：Standard edition，标准版
EE：Enterprise edition，企业版
ME：Micro edition，是为机顶盒、移动电话和PDA之类嵌入式消费电子设备提供的Java语言平台


同意协议后即可下载

打开下载好的文件，点击运行

一直点继续就行



下一步进行环境配置：
首先打开终端：

1.查看安装路径：
/Library/Java/JavaVirtualMachines/jdk1.8.0_341.jdk/Contents/Home


2.修改配置文件
sudo vim ~/.bash_profile #如果没有这个文件使用torch进行创建
#在文件中添加下面的内容，注意JAVA_HOME为刚刚查到的JDK安装路径，根据自己的进行修改！
export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk1.8.0_341.jdk/Contents/Home
export PATH=$JAVA_HOME/bin:$PATH:.
export CLASSPATH=$JAVA_HOME/lib/tools.jar:$JAVA_HOME/lib/dt.jar:.
3.使配置文件生效
source ~/.bash_profile
4.查看JDK是否安装成功
java -version
5.查看相关命令
javac



至此IDEA和JDK安装成功！
