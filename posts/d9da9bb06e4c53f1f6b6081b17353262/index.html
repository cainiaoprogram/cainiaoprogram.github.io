Android11 Launcher3调整app图标的显示顺序

比如我们的apk的包名是 com.example.test，想显示在第一个位置，如何做？

在 AlphabeticalAppsList的 onAppsUpdated 方法的最后一行

 updateAdapterItems();

之前添加如下代码：

		AppInfo temp = null ;
		for(AppInfo a :mApps){
			if(a.componentName.getPackageName().equals("com.example.test")){
				temp = a  ;
				break;
			}
		}
		if(temp != null){
			mApps.remove(temp);
			mApps.add(0,temp);
		}
