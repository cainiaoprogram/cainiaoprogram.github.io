删除

要删除已经下载的WSL（Windows Subsystem for Linux）发行版，可以通过以下步骤：

  1. 打开 PowerShell 作为管理员。

  2. 运行以下命令列出已安装的 WSL 发行版：

    wsl --list --verbose

    这将显示已安装的发行版列表，类似于：

    Windows Subsystem for Linux Distributions:
Ubuntu-20.04 (Default)
Ubuntu-18.04

  3. 使用以下命令删除特定的 WSL 发行版，比如删除名为 “Ubuntu-18.04” 的发行版：

    wsl --unregister <发行版名称>

    例如：

    wsl --unregister Ubuntu-18.04

    请确保在 <发行版名称> 处输入你想要删除的实际发行版名称。

  4. 删除 WSL 发行版后，你可以再次运行 wsl --list --verbose 命令来确认该发行版是否已被成功删除。

注意：删除发行版会删除其中的所有数据。确保在删除之前备份你重要的文件或数据。

启动

如果只有一个发行版本

  1. 使用 PowerShell：

    • 打开 PowerShell。

    • 输入以下命令启动 WSL 发行版：

      bashCopy code
wsl

  2. 使用命令提示符：

    • 打开命令提示符。

    • 输入以下命令启动 WSL 发行版：

      bashCopy code
wsl

如果系统上安装了多个 WSL 发行版，你可以在命令行中使用 wsl 命令来指定要启动的发行版。以下是一些方法：

通过指定名称启动发行版

使用 -d 或 --distribution 参数来指定要启动的发行版名称。例如：

wsl -d Ubuntu

这将启动名为 “Ubuntu” 的 WSL 发行版。将 Ubuntu 替换为你想要启动的实际发行版名称。

默认启动发行版

可以设置一个发行版作为默认发行版。启动命令行并键入以下命令，将其中一个发行版设置为默认：

wsl --set-default <发行版名称>

例如：

wsl --set-default Ubuntu

这样，在没有指定具体发行版名称时，wsl 命令将会启动默认的发行版。

通过 Windows 终端或其他终端工具选择

如果你使用 Windows 终端或其他支持 WSL 的终端工具，可以直接从菜单或设置中选择要启动的发行版。通常这些终端工具都提供了一个简单的界面来管理多个发行版。

在系统上有多个发行版时，这些方法可以帮助你方便地选择并进入特定的 WSL 发行版。
