1.服务优化

右键我的电脑–>管理–>服务和应用程序–>服务–>鼠标右击想要关闭的服务–>属性

  1. Distributed Link Tracking Client
    个人用户一般用不到，设置为手动

  2. Remote Registry
    这个服务是必停的服务之一，设置为禁用

  3. Superfetch
    I/O操作频繁的功能，如果不需要这个功能，设置为禁用

  4. Windows Defender Service
    I/O操作频繁的功能，如果不需要这个功能，设置为禁用

  5. Windows Search
    I/O操作频繁的功能，如果不需要这个功能，设置为禁用

  6. Security Center
    Windows安全中心服务，可以禁用

  7. Distributed Link Tracking Client:
    这个功能一般都用不上，完全可以放心禁用。

  8. Fax
    利用计算机或网络上的可用传真资源发送和接收传真。不用我说了吧，很明显就能看出来是和传真有关的。手动或禁用

  9. Windows Error Reporting Service
    没人喜欢错误，对你和微软而言，错误报告传送过去都没什么用。关了它。

  10. Print Spooler
    如果不需要连接打印机设备，设置为手动(不用打印机的可以关闭，不过一些软件需要此功能)

  11. Server
    如果不会共享文件或打印机，设置为手动

  12. Smart Card(默认手动)
    如果不用U盾等智能卡，设置为禁用

  13. Windows Update
    如果你不用自动更新，设置为手动

  14. HomeGroup Listener
    为家庭群组提供接收服务，该服务的默认运行方式是手动，如果你不使用家庭群组来共享图片视频及文档,那么该服务可以禁用。

  15. HomeGroup Provider
    为家庭群组提供网络服务，该服务的默认运行方式是自动，如果你不使用家庭群组来共享图片视频及文档,那么该服务可以禁用。

  16. Windows Biometric Service
    如果你没有使用生物识别设备，如指纹识别系统,该功能就可以放心禁用，否则保持默认。

  17. Windows CardSpace
    像Smart Card一样的个人标识管理，.NET Framework 3.0提供的一个WCF编程模型。一般用户可以关闭。

  18. Windows Firewall
    Windows 防火墙不用多说什么了。如果不需要就禁用。

  19. Application Layer Gateway Service
    为 Internet 连接共享提供第三方协议插件的支持。设置为禁用。

  20. Print Spooler
    如果有打印机则不需要改动

  21. Shell Hardware Detection
    用来CD和U盘，移动硬盘等移动设备的自动播放用的。设置为禁用。

  22. SSDP Discovery Service
    用于局域网上UPNP,没有可改为禁用。

  23. Terminal Service
    是远程桌面管理程序，允许多位用户控制一台计算机，个人电脑的话可以把这个服务禁用。

右键我的电脑–>属性–>高级–>性能选项设置，根据需要选择，取消某些选项后会造成一些些麻烦，如果对外观没有需求的话，可以选择 调整为最佳性能

4.安全设置

其实目前防火墙啊，各大软件的检测之类的已经做得足够完善，安全设置这块看你自己放不放心，不放心就设置设置叭

禁止建立空链接

Win+R打开运行窗口，输入regedit ,找到 “计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Lsa”中的“restrictanonymous”，双击更改数值为1

禁止管理共享

禁用c $ 、d $ 等默认共享，Win+R打开运行窗口，输入regedit ,找到
" 计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\LanmanServer\Parameters"下建立一个DOWRD数据，命名为“AUTOSHAREWKS”，设置键值为0

5.缩短系统响应时间

打开注册表编辑器：【HKEY_CURRENT_USER】-> 【Control Panel】->【Desktop】,将WaitToKillAppTimeout数值数据改成0

6.清除内存中使用不到的Dll文件

在注册表编辑器窗口，依次展开并定位到HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer，在右侧窗口空白处，单击鼠标右键，新建名为AlwaysUnloadDLL的字符串，数值设置为1

7.日常清理养护

  1. 定期碎片整理
  2. 在官网定期硬件驱动更新（看情况，这一点众说纷纭，有人说驱动要更新，有人说不用。。。。）
  3. 定期垃圾清理、安全扫描
  4. 鲁大师硬件检测，硬盘哨兵检测硬件情况：https://www.lanzous.com/i6nwupg 解压后执行“HDSentinel.exe”
  5. 定期电脑清灰（交给专业人员，小白就不要拆了），卸载无用软件（安装转件时注意捆绑安装）等
  6. 杀毒软件不要安装多于一个，其实不用也可以，电脑本身的防火墙、火绒已经够用了（腾讯、360一生之黑）
  7. C盘能不用就不用，东西及软件安装尽量放在其他盘中
  8. 不要频繁地开关机，每次关、开机之间的时间间隔应不下于30s
  9. 我个人推荐注册表定期清理，但是我以前使用的软件找不到了，大家如果要清理的话一定要找信任的软件，要不然容易出大问题
