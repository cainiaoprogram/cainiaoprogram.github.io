ubuntu系统安装好搜狗输入法后只能输入英文，无法输入中文的解决方案

写在前面

自己的电脑环境：Ubuntu20.04.5

一、问题描述

自己通过搜狗输入法官网下载deb安装包，然后按照如下指令安装搜狗输入法之后

sudo dpkg -i sogoupinyin_4.0.1.2800_x86_64.deb

并且完成相关配置。（具体方法参考：Ubuntu 18.04 安装搜狗输入法）

发现，自己只能输入英文，无法输入中文。

二、解决方法

主要是因为缺少包导致的。
安装以下两个包：

sudo apt-get install libqt5qml5 libqt5quick5 libqt5quickwidgets5 qml-module-qtquick2
sudo apt install libgsettings-qt1

然后重启搜狗输入法，就可以正常输入中英文了。
