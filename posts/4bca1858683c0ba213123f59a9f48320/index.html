linux升级pip版本

 进入python的安装路径，我这里安装在/opt/python3/在bin目录下执行

 python3 -m pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

[root@localhost bin]# python3 -m pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

升级pip 

/opt/python3/bin/python3.8 -m pip install --upgrade pip 

[root@localhost bin]# /opt/python3/bin/python3.8 -m pip install --upgrade pip

 查看是否升级成功  pip3  -V
