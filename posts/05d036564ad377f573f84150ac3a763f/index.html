mysql中报错Undeclared variable，及modify和alter区别

Undeclared variable

原因：
mysql 不支持该语句 ：

 SELECT * INTO mew_table 
 FROM old_table

不支持向还未建立的表中插入数据

解决：

CREATE TABLE new_table
(
SELECT *
FROM old__table
)

modify和alter区别

alter放在语句前面，表示插入或者修改表，对象是表；而modify修改属性，对象是表属性

数据库中下标从零开始，而且闭区间

选择字符串前 三个字符
SELECT MID(City,1,3) as SmallCity FROM Persons
