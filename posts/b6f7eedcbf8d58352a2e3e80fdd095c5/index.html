安装Anaconda时遇到的“too few arguments“问题

too few arguments:

检验安装时，出现的

usage: conda-scricy.py [-h] [-V] command ...
conda-script.py:error: too few arguments

这种情况是由于参数不对，不代表Anaconda安装出现了问题，是由于conda命令后需要加上具体的命令，如：

conda list //列出已安装的包
conda create --name myenv python=3.7 //创建一个python虚拟环境
conda activate myenv   //激活一个环境
conda install package_name//安装一个包
conda env list  //列出环境变量
conda update conda//更新conda

运行这些命令，就可以执行正确的conda命令.
