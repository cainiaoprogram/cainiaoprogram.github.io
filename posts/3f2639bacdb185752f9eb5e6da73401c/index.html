解决“ImportError: Missing optional dependency ‘pytables‘. Use pip or conda to install pytables.”问题

我用的是python3.8，我在论坛里看到有人提到python3.8.2版本pip里没有pytables。

所以是下载tables解决的：

pip install tables

我测试过使用conda下载pytables也可以：

conda install pytables
