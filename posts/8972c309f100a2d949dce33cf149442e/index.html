概述

如果你想拥有一个属于自己的路由器操作系统，Openwrt是一个不错的选择，本文描述了如何编译一个Lede系统，并附 树莓派4B的 config文件

src-git telephony <https://git.openwrt.org/feed/telephony.git>

#src-link custom /usr/src/openwrt/custom-feed

拉取插件并更新

./scripts/feeds update -a
./scripts/feeds install -a

修改默认IP(可选，根据自己需要)

sed -i 's/192.168.1.1/192.168.3.1/g' package/base-files/files/bin/config_generate

这里修改后，默认ip：192.168.3.1，默认密码：password


修改默认主题

1.删除lede的luci源中的argon旧版本主题(需先feeds update):

rm -rf feeds/luci/themes/luci-theme-argon

2.拉取最新的18.06分支的argon主题源码到 package/downloads/luci-theme-argon :

git clone -b 18.06 <https://github.com/jerrykuku/luci-theme-argon.git> package/downloads/luci-theme-argon

3.修改lede的默认主题,位置 feeds/luci/collections/luci/Makefile :

sed -i 's/luci-theme-bootstrap/luci-theme-argon/' feeds/luci/collections/luci/Makefile

生成系统配置

有两种方法

  1. 使用 make menuconfig 根据自己需要生成配置
  2. 使用修改好的.config文件，见附录

编译系统

下载 dl 库，编译固件 （-j 后面是线程数，第一次编译推荐用单线程）

make download -j8
make V=s -j1

二次编译

cd lede
git pull
./scripts/feeds update -a
./scripts/feeds install -a
make defconfig
make download -j8
make V=s -j$(nproc)
