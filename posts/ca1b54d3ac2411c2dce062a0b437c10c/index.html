linux之jq命令

jq命令用于linux命令行对json进行处理

参数

option

-r：去掉字符串的引号'"'

例子

tt.json文件如下：

[
    {
        "metric": "httpcode",
        "tags": {
            "cluster": "tt",
            "domain": "www.baidu.com"
        }
    },
    {
        "metric": "httpcode",
        "tags": {
            "cluster": "tt",
            "domain": "www.pingan.com"
        }
    },
    {
        "metric": "httpcode",
        "tags": {
            "cluster": "tt",
            "domain": "www.dy.com"
        }
    }
]

# jq命令后面一定不能忘了加引号
lxm@mypc  ~/Documents/image  cat tt.json| jq -r '.[].tags.domain'
www.baidu.com
www.pingan.com
www.dy.com
