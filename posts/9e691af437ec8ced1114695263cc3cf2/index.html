SQL语言实现MYSQL表的复制

create table table_name select * from table_name  。这种复制表，显然只是复制了表的数据，表的结构，包括索引等一些，都没有复制。

所以复制表时要先复制表结构，

create table table_name like table_name;

insert into table_name select * from table_name;

这样，才能较完整的复制一张表
