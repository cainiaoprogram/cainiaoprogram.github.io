docker 安装操作(Mac版)

安装docker(两种方式)

方式一:命令下载安装

brew cask install docker

方式二:网页直接下载

下载网址1：https://download.docker.com/mac/stable/Docker.dmg
下载网址2：https://download.docker.com/mac/edge/Docker.dmg

注意：下载安装时可能需Mac账户密码

安装成功后菜单栏会出现docker的图标，首次使用docker时，可能需要Mac账户密码，视情况而定。同时打开终端执行docker命令(下列命令仅用于测试docker是否安装成功，更多的命令可以参考菜鸟https://www.runoob.com/docker/docker-command-manual.html，docker命令大全):

1.docker --version(查看版本号)
2.docker images(列出本地镜像)
3.docker ps(列出容器)

docker镜像加速(新版本的docker图形化工具与教程中不太一样)

操作如下图所示

"registry-mirrors": [
    "https://registry.docker-cn.com"
  ],

添加完，点击“Apple&Restart”按钮

在终端执行命令，显示如下图所示

docker info

OK，完成安装！
