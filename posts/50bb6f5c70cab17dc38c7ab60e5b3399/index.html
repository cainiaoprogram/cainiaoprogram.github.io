vscode 在debug模式给被调试程序传递环境变量

在debug时，需要给程序传递特殊的环境变量，而又不想修改系统的环境变量，使用以下方法。对于不同的编程语言，其使用属性的名称不一样。

1. c/c++

使用environment 关键字，在配置文件中增加以下配置。注意格式。

 "environment": [{"name": "ENV_TEST", "value": "1"},
                 {"name": "TEST1", "value": "23"}
            ],

2. Python

使用 env 关键词去设置传递的环境变量，注意格式。

  "env": {
          "ENV_TEST":"1",
          "ENV1":"12",
	}
