原生 select 标签选择后的边框修改

左图是初始化边框样式，右图是选择后，自带了粗边框。太丑了....要去除，修改如下：

select:focus-visible {
    /* outline: -webkit-focus-ring-color auto 1px; 这是谷歌浏览器的原生样式*/
    outline: none;
}
