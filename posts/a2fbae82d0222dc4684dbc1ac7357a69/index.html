Java之多线程下载网络图片

一、下载工具类库

commons IO包下载地址

解压bin目录下的jar包放到idea新建lib目录中

进行加入库操作

二、编写下载器

用的是FileUtils.copyURLToFile

class WebDownLoader{
    //下载方法
    public void downloader(String url, String name){
        try {
            FileUtils.copyURLToFile(new URL(url), new File(name));
        } catch (IOException e) {
            e.printStackTrace();
            System.out.println("IO异常,downloader出现异常");
        }
    }

三、多线程的实现

线程是继承Thread类重写run方法(下载图片的执行体)

public class TestThread2 extends Thread{
    private String url; //网络地址
    private String name;//保存文件名

    public TestThread2(String url, String name){
        this.url = url;
        this.name = name;
    }

    @Override
    public void run() {
      super.run();
    }
}

四、总的代码

package com.kuang.demo1;

import org.apache.commons.io.FileUtils;

import java.io.File;
import java.io.IOException;
import java.net.URL;

/************************************
 * @PACKAGE : com.kuang.demo1
 * @Author    :  叫我老伯
 * @Date    : 2023/3/2 13:12
 * @Desc    :
 ***********************************/
//实现多线程下载图片
public class TestThread2 extends Thread{
    private String url; //网络地址
    private String name;//保存文件名

    public TestThread2(String url, String name){
        this.url = url;
        this.name = name;
    }

    @Override
    public void run() {
        WebDownLoader webDownLoader = new WebDownLoader();
        webDownLoader.downloader(url, name);
        System.out.println("下载的文件名为" + name);
    }

    public static void main(String[] args) {
        TestThread2 testThread2 = new TestThread2("https://img-blog.csdnimg.cn/ecc554ff1bfb4feb9b1a4a4444c7cf8c.png", "2.png");
        TestThread2 testThread3 = new TestThread2("https://img-blog.csdnimg.cn/142817d142c64243a01169f67eacc154.png", "3.png");
        TestThread2 testThread4 = new TestThread2("https://img-blog.csdnimg.cn/997601f9eea0409fac770af5a2863c57.png", "4.png");

        testThread2.start();
        testThread3.start();
        testThread4.start();
    }
}

//下载器
class WebDownLoader{
    //下载方法
    public void downloader(String url, String name){
        try {
            FileUtils.copyURLToFile(new URL(url), new File(name));
        } catch (IOException e) {
            e.printStackTrace();
            System.out.println("IO异常,downloader出现异常");
        }
    }
}

注意：并不是按着执行顺序来下载的，而是同时下载的
