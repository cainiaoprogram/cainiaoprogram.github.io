【C语言】从键盘任意输入一个3位整数，编程计算并输出它的逆序数（忽略整数前的正负号）。

C语言实战题目：从键盘任意输入一个3位整数，编程计算并输出它的逆序数（忽略整数前的正负号）。例如，输入-123，则忽略负号，由123分离出其百位1、十位2、个位3，然后计算3*100+2*10+1 = 321，并输出321。

**输入格式要求："%d" 提示信息："Input x:"

**输出格式要求："y = %d\n"

程序运行示例如下：

Input x:-123

y = 321

#include <stdio.h> 
#include<math.h>
int main()
{
    int a, b, c, sum, num;
    scanf("%d", &num);
    printf("Input x:");
    num = fabs(num);
    a =num% 10;
    b =num% 100 / 10;
    c =num/ 100;
    sum= (a * 100 + b * 10 + c);
    printf("y = %d\n", sum);
    return 0;
}
