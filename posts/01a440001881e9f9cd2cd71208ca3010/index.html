移动硬盘无法在ubuntu显示

sudo fdisk -l (找到移动硬盘路径)
sudo mkdir /media/mq/jorbo （这里填自己的用户名（mq）及想创建的名字（jorbo））
sudo mount -t exfat /dev/sda1** /media/mq/jorbo （移动硬盘路径 自己创建的路径）
若报错

fuse: mountpoint is not empty fuse: if you are sure this is safe, use the ‘nonempty’ mount option

sudo mount -t exfat -o nonempty /dev/sda1** /media/mq/jorbo
