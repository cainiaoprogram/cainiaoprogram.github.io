windows下ConcurrentLogHandler因锁机制导致的卡死问题

用concurrent-log-handler替换ConcurrentLogHandler可解决windows因锁机制导致的卡死问题。

即pip install ConcurrentLogHandler 改为pip install concurrent-log-handler.
