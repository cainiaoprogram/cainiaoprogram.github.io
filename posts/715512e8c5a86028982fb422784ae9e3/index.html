一、题目

猜数字，数字随机产生，共三次机会:
猜错有错误提示，猜对bingo并结束

二、过程

1.代码

代码如下：

import random
num=int(input("你猜猜我想的是什么数字呀？（悄悄给个提示，是个正整数哦）\n三次机会，把握好哦\n第1次猜测："))
secret=random.randint(1,10)
i=1
while i<=3:
    if num!=secret:
        if num<secret:
            print("猜小了呢~")
        else:
            print("哎呀呀，猜大啦！")
    else:
        print("BINGO!")
        break
    i = i + 1
    if i!=4:
        print("第{0}次猜测：".format(i))
        num =int(input())

    else:
        print("哦~答案是{0}，都没猜对呢/(ㄒoㄒ)/~~".format(secret))
print("游戏结束！")
