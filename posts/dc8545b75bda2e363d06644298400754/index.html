CentOS 升级 Camke

下载源码

官网地址：https://cmake.org/files/

wget https://cmake.org/files/LatestRelease/cmake-3.15.3.tar.gz

解压

tar -vxf cmake-3.15.3.tar.gz

安装

cd cmake-3.15.3
./bootstrap --prefix=/usr/local
gmake
gmake install

查看编译后的版本

/usr/local/bin/cmake --version

移除原来的cmake版本

yum remove cmake -y

新建软连接

ln -s /usr/local/bin/cmake /usr/bin/

查看最终版本

cmake --version
