Python：猜数字 （输入一个数字，打印比较的结果：“猜错了”、“猜对了”、“答对了”）

题目三

代码

answer = 88
while True:
    your_answer = int(input("请输入你的答案："))
    if your_answer > answer:
        print("太大了")
    elif your_answer < answer:
        print("太小了")
    else:
        print("答对了")
        break
