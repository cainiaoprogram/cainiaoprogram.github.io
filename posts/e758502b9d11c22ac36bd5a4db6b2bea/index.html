（二）VMware搭建华为FusionCompute6.5.1，可正常使用 —— 开启X11转发，配置桥接，安装FusionCompute

1.开启X11转发

1.1 编辑ssh配置文件

vim /etc/ssh/sshd_config

X11Forwarding yes        #设置为yes 

systemctl restart sshd

1.2 在PC端直接开启虚拟系统管理器 

 2.配置桥接

2.1 创建桥接，使其与物理网卡对接

yum install -y bridge-utils                # 依赖包，已安装

ifconfig                                            # 查看网卡名

brctl addbr br0                                # 创建网桥

ifconfig br0 192.168.80.100 netmask 255.255.255.0        # 给个IP临时激活

nmcli connection modify br0 ipv4.addresses 192.168.80.100/24 ipv4.gateway 192.168.80.2 ipv4.dns 114.114.114.114 connection.autoconnect yes        # 生成br0网卡文件

vim /etc/sysconfig/network-scripts/ifcfg-ens33

TYPE=Ethernet
NAME=ens33
DEVICE=ens33
ONBOOT=yes
BRIDGE=br0                # 将网卡对接br0

nmcli co reload

nmcli co up ens33

nmcli co up br0

3.创建虚拟机

3.1 上传ISO文件

链接：https://pan.baidu.com/s/1CbXQsJ8e6TvJZsqocc1lRw?pwd=bc6u 
提取码：bc6u

3.2 创建虚拟机

选2020

 

没licenses的集群最多6个CPU，这边配置不足只能一台CNA就全给了(记得是这样说的，但是给十个好像也能用。。)

建个磁盘文件

qemu-img create -f qcow2 CNA.qcow2 240g  -o preallocation=metadata

 

选择自定义 

3.3 自定义虚拟机

CPU修改为        host-passthrough 

磁盘总线改为 SATA

 网卡型号改为e1000e

 保存，安装

 3.4 开始安装CNA

略                不会装机搭建什么FC        遇到按不动切下屏解决

 3.5 开始安装VRM

用virt-install命令的方式来安装

依赖包yum -y install virt-install virt-viewer        # 提供安装        提供vnc界面

安装命令

qemu-img create -f qcow2 VRM.qcow2 500g  -o preallocation=metadata

virt-install --name VRM --memory 6144 --vcpus 4 --cdrom /iso/FusionCompute\ 6.5.1_VRM.iso --disk /data/VRM.qcow2 --network bridge=br0 --osinfo detect=no,name=linux2020 --boot cdrom        # 跟着是敲不出来的
