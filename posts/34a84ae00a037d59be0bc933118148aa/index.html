kubelet启动报错cannot find network namespace for the terminated container

1.报错：

使用journalctl -xefu kubelet命令查看kubelet日志发现如下报错：

cannot find network namespace for the terminated container

2.解决办法：

# docker system prune

# systemctl restart kubelet

docker system使用说明：
