numpy将矩阵中小于0的值换成0的实例

a is a matrix or an array

  1. np.maximum(a, 0)
  2. (a+np.abs(a))/2
  3. a[a<0]=0
  4. np.where(a > 0, a, 0) a中大于0的用a中的元素不变，小于0的就用0补上
  5. b = np.float32(a<=0)-1 np.float32(a<=0)会将小于0的变成1，所以要换成0需要减去1
