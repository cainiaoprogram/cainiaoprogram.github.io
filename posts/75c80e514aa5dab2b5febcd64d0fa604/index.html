windows server backup 作业执行失败后的恢复

当Windows server backup执行作业失败，想要停止比较麻烦，点击取消备份进度条无限循环中……

执行命令wbadmin stop job也是永远等待的状态。

可以通过停止Block Level backup engine service来解决。

 

重新启动此服务后，不用重新启动服务器，就可以重新来配置失败的作业了。
