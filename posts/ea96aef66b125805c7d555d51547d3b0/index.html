SAS关于自定义format的小知识点

proc format lib=work;
value agefmt
	11-13 ='小'
	14-15='大';
run;
data class2;
	set class;
	format age agefmt.;
run;

要注意格式语句中小小的点，没点SAS会认为是变量。加上点会认为是格式。
