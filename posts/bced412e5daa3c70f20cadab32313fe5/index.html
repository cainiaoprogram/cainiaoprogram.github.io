人生的第一篇CSDN文章（第一个C程序）

作为程序员人生写的第一个C程序：

#include<stdio.h> //编译预处理命令
int main()        //定义函数的类型：整型（int）
{                 //函数开始的标志
    printf("Hello world!\n");  //输出内容 ，'\n'换行符
}                 //函数结束的标志

3非常感谢CSDN这个平台和共同学习和分享的程序员……

4最后，我会在以后的日子里努力学习C语言，并把我学到的知识好好总结下来，和大家一起学习和交流，共同进步。💪🐵🐒🧐
