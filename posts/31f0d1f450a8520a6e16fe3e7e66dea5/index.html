Latex之figure的嵌套：Subfig与subfigure

在论文中经常有一个figure内含有其他小的figure,这时就要使用一些宏包。
subfigure是比较老的了，目前通用的是subfig,两个包在使用的时候用法不同，千万不要混淆了。不然莫名其妙的报错根本不知道是怎么回事。

例：
subfig

\begin{figure}[tbp]
	\centering
	\subfloat[Arabic numerals]{\label{fig:a}\includegraphics[width=1in]{placeholder}}\quad
	\subfloat[Arabic numerals]{\label{fig:b}\includegraphics[width=1in]{placeholder}}\\	
	\caption{Capital Roman numerals.}
\end{figure}

subfigure

\begin{figure} \centering 
\subfigure[figure 1 title.] { \label{fig:a} 
\includegraphics[width=0.8\columnwidth]{fig1.eps} 
} 
\subfigure[figure 2 title.] { \label{fig:b} 
\includegraphics[width=0.8\columnwidth]{fig2.eps} 
} 
\caption{ general title. } 
\label{fig} 
\end{figure}
