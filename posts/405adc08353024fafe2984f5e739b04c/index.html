C2360***的初始化操作由case标签跳过的解决办法

C2360***的初始化操作由case标签跳过：

原报错代码：
								switch (***)
								{
									case 1:		
										int ret_1 = 1;
										break;
									case 2:
										int ret_2 = 2;
										break;
									default:
										int ret_3 = 3;
										break;	
								}

方法一：
case后内容加{}：

修改后代码：
								switch (***)
								{
									case 1:		
										{		
										int ret_1 = 1;
										break;
										}
									case 2:
										{
										int ret_2 = 2;
										break;
										}
									default:
										{
										int ret_3 = 3;
										break;	
										}
								}

方法二：
在switch之外初始化（这里就不举例了）
