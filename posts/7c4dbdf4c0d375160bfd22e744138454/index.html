vue框架中使用fontAwesome图标

1、第一步：安装fontAwesome图标依赖

npm install fontAwesome  --save

2、第二步：在main.js中引入fontAwesome图标依赖

import 'font-awesome/css/font-awesome.css';

3、第三步：使用图标

 <i class="fa fa-address-book-o" aria-hidden="true"></i>
