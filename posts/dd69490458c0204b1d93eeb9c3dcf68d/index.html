intouch的服务器修改密码后,intouch登录设置

1：定义标记

Usernameinput 内存消息 (用户名)

Pwdinput 内存消息 (旧密码) Pwdqr 内存消息 (新密码) 2：写语句

登录语句

IF usernameinput=="username" THEN

IF pwdinput=="pwd" THEN

ELSE

Show "E1";

ENDIF;

ENDIF;

IF usernameinput=="nbps" THEN

IF pwdinput=="nbps" THEN

ELSE

Show "E1";

ENDIF;

ENDIF;

usernameinput="***";

pwdinput="**********";

HideSelf;

其中Show "E1";

E1为intouch窗口

ENDIF;

HideSelf;

ENDIF;
