发现底部出现横向滚动条，

 问题大概在于 
<el-scrollbar></el-scrollbar> 
上的 height: 100% 与 margin-bottom 同时显示

 直接用 overflow-x:hidden  隐藏：

.el-scrollbar__wrap {
    overflow-x: hidden;
}

PS：通过阅读源码，可以看到暴露的几个属性

props: {
    native: Boolean,    // 使用原生滚动条
    wrapStyle: {},      // 包裹层
    wrapClass: {},
    viewClass: {},      // 内容层
    viewStyle: {},
    noresize: Boolean,  // 如果 container 尺寸不会发生变化，最好设置它可以优化性能
    tag: {              // 编译后生成的标签，默认 `div`
      type: String,
      default: 'div'
    }
  },

所以，前面的代码可以插入这些属性

<div style="height:100%">
  <el-scrollbar
     :native="false" 
     wrapStyle="" 
     wrapClass="" 
     viewClass="" 
     viewStyle="" 
     noresize="false" 
     tag="section">
    <router-view/>
  </el-scrollbar>
 </div>
