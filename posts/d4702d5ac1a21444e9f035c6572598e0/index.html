qps tps rps 区别

qps

query per second
每秒查询数
例如一次 sql 查询就是一次 query

tps

transaction per second
每秒事务数
例如一次事务 sql 包含 3 个语句, 那么其实是一次事务

rps

request per second
每秒请求数
例如请求一次 http 接口, 就是一次 request
