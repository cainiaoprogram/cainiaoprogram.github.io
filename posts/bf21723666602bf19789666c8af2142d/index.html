CentOS7 使用 yum 安装 chrome

1、下载 rpm 包

搜索 “Google Chrome for Linux 下载” 得到:

例如：http://down.tech.sina.com.cn/page/43719.html

2、用 yum 安装

yum localinstall google-chrome-stable_current_x86_64.rpm

yum会自动分析依赖，完成安装
