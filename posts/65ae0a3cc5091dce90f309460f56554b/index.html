东方博宜OJ 1264 - 【入门】4位反序数

题目描述

设N是一个四位数，它的9倍恰好是其反序数，求N。反序数就是将整数的数字倒过来形成的整数。例如：1234的反序数是4321。

输入： 无
输出： 输出N这个四位数
来源： 简单循环
标签： 简单循环 
#include<stdio.h>
using namespace std;
int main()
{
	int i,j,k,qw,bw,sw,gw;
	for(i=1000;i<10000;i++)
	{
		qw=i/1000,bw=i%1000/100,sw=i%100/10,gw=i%10,j=gw*1000+sw*100+bw*10+qw;
	    if(j==i*9) printf("%d",i); 
	}
	return 0;
}
