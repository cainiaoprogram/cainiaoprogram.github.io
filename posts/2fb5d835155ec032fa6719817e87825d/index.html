【flink】项目初始化

mvn archetype:generate                \
  -DarchetypeGroupId=org.apache.flink   \
  -DarchetypeArtifactId=flink-quickstart-java \
  -DarchetypeVersion=1.17.1 \
  -DgroupId=com.example \
  -DartifactId=paimon \
  -Dversion=1.0.1 \
  -Dpackage=com.example \
  -DinteractiveMode=false

目录结构

➜  ~ tree paimon
paimon
├── pom.xml
└── src
    └── main
        ├── java
        │   └── com
        │       └── example
        │           └── DataStreamJob.java
        └── resources
            └── log4j2.properties
