Windows cmd net 命令

一、net view 用法 

1、查看局域网内计算机

net view

2、主名机查 MAC 地址

nbtstat -a hostname

3、查看 IP 与 MAC 地址映射表 来获取 IP

arp -a

4、根据 IP 查主机名

nbtstat -A IP

二、net use 用法

1、建立非空连接 

net use \\10.3.0.39\ipc$ "passwd" /user:"username"

2、建立连接后，并映射 z 盘

net use z: \\10.3.0.39\Public "passwd" /user:"username"

3、建立连接后，映射 z 盘，并修改文件夹名

net use z: \\10.3.0.39\Public "passwd" /user:"username"

# 修改文件夹名
reg add "HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\MountPoints2\##10.3.0.39#Public" /v "_LabelFromReg" /d "Public" /f

 4、 查看连接

net use


# 会记录新的网络连接。


状态       本地        远程                      网络

-------------------------------------------------------------------------------
OK           Z:        \\10.3.0.39\Public        Microsoft Windows Network
OK                     \\10.3.0.39\ipc$          Microsoft Windows Network
命令成功完成。

5、删除带映射的连接

net use z: /del

 三、net share 用法

默认共享是系统安装完毕后就自动开启的共享，也叫管理共享，常被管理员用于远程管理计算机。在Windows 2000/XP及其以上版本中，默认开启的共享有"c$"、"d$"、"admin$"、"ipc$"等，我们可以在"运行"对话框中输入"\\计算机名\盘符$"对这些资源进行访问。

1、查看本机开启的共享

net share

共享名       资源                            注解

-------------------------------------------------------------------------------
C$           C:\                             默认共享
IPC$                                         远程 IPC
ADMIN$       C:\Windows                      远程管理
命令成功完成。

四、net view 提示 6118 错误解决方法

1、win+R ,输入services.msc 开启服务：Server ，WorkStation，computer Browser

2、添加 SMB 功能

进入：控制面板 – 程序和功能 – 启用或关闭 windows 功能，选中 SMB 1.0/CIFS 文件共享支持

3、重启生效
