MobaXterm连接Ubuntu系统

新安装的Ubuntu系统直接用MobaXterm连不上，我们可以先打开终端看Linux的IP，我们设置的网络连接方式为net模式，在终端中输入如下命令

ip addr

需要新安装ssh才可以用MobaXterm连接，执行如下命令

apt install openssh-server

 禁用防火墙后即可用MobaXterm连接

ufw disable
