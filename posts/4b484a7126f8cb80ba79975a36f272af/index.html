一维数组（数组）

数组

例题：输入n个学生的成绩，求他们的平均成绩，并求出成绩大于平均分的人数

先考虑求平均成绩，代码怎么写？

#include <cstdio>
int main() {
    int n, score, sumn = 0;
    double aver;
    scanf("%d", &n);
    for (int i = 0; i < n; ++i) { //边读边累加求和
        scanf("%d", &score);
        sumn += score;
    }
    aver = (double)sumn / n;
}

数组的基础代码就是这样的；

#include <bits/stdc++.h>
using namespace std;
int main(){
    int n,i,a[100];//n代表的是输入的个数
    //i用于for()循环
    //a[100]就是数组 
    scanf("%d",&n);//输入n（n代表要输入n个数） 
    for(i=1;i<=n;i++){//输入范围 
        scanf("%d",&a[i]);//for()循环输入 
    }
    printf("%d",a[n]);//输出第n个数组 
    return 0;
}

数组的概念

数组是连续存储数据的集合。

组成数组的每个数据称为数组的元素数组的每个元素都属于同一种数据类型

一个数组的所有元素在内存中的存储位置是连续的！

定义

int a[10]; //定义能存储10个整形变量的数组，下标是:0~9
char c[100];//定义能存储100个字符变量的数组，下标是:0~99
double f[5];//定义能存储5个双精度实数变量的数组，下标是:0~4

数组的初始化

数组的初始化可以在定义时一并完成，有多种形式：

顺序指定全部元素的初始值

int a[5] = {1,2,3,4,5};

顺序指定部分元素的初始值

int x[10] = {0,1,2,3,4};//该方法仅对数组的前5个元素依次进行初始化，其余值为0。

对数组元素全部初始化为0，可以简写为：{}或

int a[5]={};//将数组a的5个元素都初始化为0。

初始化时不指定数组元素的个数

int a[]={1, 2, 3};//该方法会定义一个长度为3的数组，每个元素初始值依次为1, 2, 3。
