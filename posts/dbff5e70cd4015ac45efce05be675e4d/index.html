常量与变量 

• Objective-C中声明常量使用关键字const：

• const double PI = 3.141592654;

• Objective-C中变量可以分为成员变量、局部变量和全局变量。 

int gCounter;//全局变量
@interface MyObject : NSObject {
      int counter;//成员变量
}@end
@implementation MyObject
-(void) print {
int cter = 0;//局部变量
      NSLog(@"%i", cter );
}@end

运算符与表达式

运算符可以分成如下几种:

•算术运算符,+,―,*,/,%,++,――

•关系运算符,>,<,>=,<=,==,!=

•布尔逻辑运算符,!,&&,||

•位运算符,&,|,^,~ ,>>,<<

•赋值运算符,+=,―=,*=,/=

•条件运算符,? :
