Camunda适配达梦数据库

​ 只需要指定数据库类型为oracle即可，具体application.yml配置如下：

camunda:
  bpm:
    webapp:
      index-redirect-enabled: false
    admin-user:
      id: demo
      password: camunda
    history-level: full
    auto-deployment-enabled: true
    authorization:
      enabled: true
    database:
      type: oracle
