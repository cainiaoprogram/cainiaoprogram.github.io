vbs进阶——条件分支

条件分支，是一个特别基础的东西，对于像我一样热爱C++的人，就是一些if,else,case之类的东西，下面我们要讲的vbs的条件分支也差不多一样.

基础代码：

if 条件 then 做的事

注意：一定不能分行，否则就会编译错误！

else

if 条件 then 做的事
else 不满足需要做的事

3个及以上的条件

if 条件 then 做的事
else if 条件 then 做的事
else if 条件 then 做的事

就这样两个单词:

if else

却能衍生出无限种可能，这就是条件分支！
