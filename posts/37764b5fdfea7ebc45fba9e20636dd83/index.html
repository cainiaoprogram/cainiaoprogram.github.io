uniapp使用第三方字体

下载第三方字体链接
https://www.100font.com/

1.下载下来将字体引入项目my-font.ttf（自定义命名）

  1. 在app.vue
    @font-face { font-family: my-font; src: url('~@/static/my-font.ttf'); }
    3.使用
    .text{ font-family: my-font }
